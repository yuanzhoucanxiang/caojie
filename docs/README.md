# caojie 游戏策划案

> 核心概念：县城出生的孩子，幼年被送到茂名农村与祖父母生活。通过事件选择塑造属性与性格，经历童年→少年→青年的人生轨迹。
>
> 游戏类型：2.5D 横板叙事 + 模拟经营 + 角色扮演
> 引擎：Godot 4.6.2

## 设计原则

- **地域真实感**：所有角色、场景、事件都以茂名真实生活为基础
- **选择驱动**：玩家通过对话/行动选择塑造角色性格，没有"正确答案"
- **最小可行原型优先**：每个阶段先验证核心玩法再扩展

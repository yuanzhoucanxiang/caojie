# 09 - 时间系统

## 时间单位

| 单位 | 说明 | 对应游戏内容 |
|------|------|-------------|
| **天** | 最小的可操作时间单位 | 一天的活动流程 |
| **季节** | 每天属于一个季节 | 不同季节触发不同事件池 |
| **阶段** | 童年/少年/青年 | 游戏的核心结构 |

---

## 一天的时间段

| 时段 | 时间 | 典型活动 |
|------|------|---------|
| 早晨 | 6:00-8:00 | 起床、吃早饭、和家人对话 |
| 上午 | 8:00-12:00 | 自由探索、触发事件 |
| 中午 | 12:00-14:00 | 吃午饭（可选快速推进） |
| 下午 | 14:00-18:00 | 自由活动（干农活/赶集/探索） |
| 傍晚 | 18:00-20:00 | 回家吃饭、和家人闲聊 |
| 夜晚 | 20:00-22:00 | 自由活动或睡觉推进时间 |

---

## 季节系统

茂名地处热带亚热带，季节特征：

| 季节 | 特征 | 事件池 |
|------|------|--------|
| **春季** (3-5月) | 温暖、潮湿、年例在正月 | 年例事件、荔枝花季 |
| **夏季** (6-8月) | 漫长、炎热、台风季 | 干农活、台风事件、放暑假 |
| **秋季** (9-11月) | 短暂、凉爽 | 荔枝/龙眼收获、学校开学 |
| **冬季** (12-2月) | 短暂、较冷 | 过年、寒假、写作业 |

# 03 - 核心循环

## 玩家每一轮做什么

游戏的时间单位是 **"一天"**。玩家在一天中的流程：

```
起床 → 白天活动（探索/对话/任务）→ 晚上 → 睡觉 → 进入下一天
```

### 一天的结构

| 时段 | 可做的事 |
|------|----------|
| **早晨** | 起床、和祖父母一起吃早饭（固定事件，简单对话） |
| **上午** | 自由活动：出门探索、找 NPC 对话、触发事件 |
| **下午** | 自由活动：可选择干农活/上学/做其他事 |
| **傍晚** | 回家吃饭、和家人聊几句 |
| **夜晚** | 睡觉前可看天花板沉思（属性小幅度变动）、或触发夜间事件 |

### 核心循环图

```
探索场景
    │
触发事件（对话/任务）
    │
做出选择 → 属性变化
    │
事件结束 → 解锁后续事件/关系变化
    │
继续探索 / 推进时间（到下一天或下一时段）
```

---

## 推进时间

- 事件结束后自动推进到下一个时间段
- 也可手动"回屋休息"推进时间
- 某些事件之间需要等待 N 天

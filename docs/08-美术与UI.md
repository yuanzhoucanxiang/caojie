# 08 - 美术与 UI

## 视觉风格

**2.5D 高精度像素美术**，参考《Until Then》《The Last Night》《REPLACED》。

### 核心视觉特征

| 元素 | 风格 |
|------|------|
| **像素精度** | 中高精度（16-32px 核心元素） |
| **色彩** | 暖色系为主（砖红、土黄、暖绿），保留粤西热带感 |
| **光影** | 环境光为主，有柔和阴影 |
| **图层** | 背景-中景-前景 多层视差 |
| **UI** | 简洁、半透明、非侵入式 |

### 色板建议

| 用途 | 色值 | 感受 |
|------|------|------|
| 天空蓝 | #4a90d9 | 开阔、清新 |
| 草地绿 | #5a8a3c | 茂名植被感 |
| 砖墙红 | #b5651d | 茂名红砖房 |
| 老瓦灰 | #7a7a6e | 旧屋、岁月感 |
| 水泥灰 | #a0a098 | 县城新建筑 |

## UI 布局

### 游戏中的 UI

```
┌──────────────────────────────┐
│ [NPC 名字]                    │
│                              │
│ ┌──────────────────────────┐ │
│ │ 对话内容区                │ │
│ │                          │ │
│ └──────────────────────────┘ │
│ ┌──────────────────────────┐ │
│ │ 选项1                      │ │
│ │ 选项2                      │ │
│ └──────────────────────────┘ │
└──────────────────────────────┘
```

- 对话框在画面底部
- 选项用按钮样式，点击或键盘选择
- 事件完成时显示属性变化提示（右上角小字）

# 08 - 美术与 UI

## 视觉风格

**2.5D 高精度像素美术**。

### 参考游戏

| 游戏 | 参考点 |
|------|--------|
| **Ori 系列** | 多层视差滚动、精致的自然场景、前景遮挡物营造景深 |
| **歧路旅人（HD-2D）** | 像素角色+3D景深效果、远处虚化、光影氛围 |
| **REPLACED** | 电影感光影、暗色调场景中的局部打光、像素精度 |
| **直到那时（Until Then）** | 日常生活场景的温暖质感、人物情感表达、2.5D像素叙事 |
| **爱氏物语（Eastward）** | 像素角色表情差分、对话时的生动演出、温暖像素画风 |

### 核心视觉特征

| 元素 | 风格 |
|------|------|
| **像素精度** | 中高精度（16-32px 核心元素） |
| **色彩** | 暖色系为主（砖红、土黄、暖绿），保留粤西热带感 |
| **光影** | 环境光为主，有柔和阴影 |
| **图层** | 背景-中景-前景 多层视差 |
| **UI** | 极简、无常驻 HUD、漫画框式对话 |

### 色板建议

| 用途 | 色值 | 感受 |
|------|------|------|
| 天空蓝 | #4a90d9 | 开阔、清新 |
| 草地绿 | #5a8a3c | 茂名植被感 |
| 砖墙红 | #b5651d | 茂名红砖房 |
| 老瓦灰 | #7a7a6e | 旧屋、岁月感 |
| 水泥灰 | #a0a098 | 县城新建筑 |

## UI 设计原则

- **无常驻 HUD**：画面保持干净，不显示血条、地图、属性等任何常驻信息
- **属性不主动提醒**：属性变化不会弹出通知，玩家需打开菜单自行查看
- **漫画框式对话**：对话以漫画气泡的形式在角色旁边展现，非底部对话框
- **表情差分演出**：对话时展示角色的像素表情差分，参考《爱氏物语》

## 对话演出系统

### 漫画框式对话

对话不是传统的底部对话框，而是像漫画一样在角色旁边出现气泡框，配合角色表情差分。

**平时画面（无对话）：**
```
┌────────────────────────────────────────┐
│                                        │
│           （完全干净的场景画面）          │
│                                        │
│                                        │
│          外婆            主角           │
│                                        │
└────────────────────────────────────────┘
```
- 画面无任何 UI 元素
- 靠近可互动物体时出现小提示（可选）

**对话画面：**
```
┌────────────────────────────────────────┐
│                                        │
│    ┌──────────────────────┐            │
│    │ ┌────┐               │            │
│    │ │ :) │ 乖孙，过来帮   │            │
│    │ └────┘ 奶奶择菜。     │            │
│    └──────────────────────┘            │
│         外婆            主角            │
│                                        │
│    ┌──────────┐  ┌──────────────┐      │
│    │好的奶奶！│  │我想出去玩…… │      │
│    └──────────┘  └──────────────┘      │
└────────────────────────────────────────┘
```
- 对话气泡从说话角色旁边冒出
- 气泡内包含角色表情差分 + 对话文本
- 选项也是漫画风格的小气泡，非传统按钮

### 对话框类型

| 类型 | 样式 | 适用场景 |
|------|------|---------|
| **普通气泡** | 小框，角色头顶或身旁 | 日常对话、短句 |
| **放大框** | 框更大，文字更多 | 长段对话、讲故事 |
| **特写框** | 镜头拉近，半身表情占画面 | 重要剧情、情感高潮 |
| **内心框** | 异形框（云朵形/锯齿形） | 主角内心想法 |

### 选项呈现

- 选项不是底部按钮列表，是漫画风格的选择气泡
- 每个气泡一句话，从主角旁边冒出
- 用方向键或点击选择
- 选中时气泡高亮放大

## 角色表情差分

每个角色需要一套像素表情差分，在对话时展示，参考《爱氏物语》的风格。

### 基础表情列表

| 表情 | 触发场景举例 |
|------|------------|
| **普通** | 日常对话、没有特别情绪 |
| **微笑** | 开心、欣慰、温馨时刻 |
| **大笑** | 搞笑场景、被逗乐 |
| **皱眉** | 不满、担心、思考 |
| **生气** | 骂人、发飙（舅舅、小凤） |
| **难过** | 离别、委屈、心疼 |
| **惊讶** | 意外事件、震惊 |
| **害羞** | 被夸、不好意思（小英常见） |
| **无语** | 阿宝又闯祸了、无奈 |

### 主要角色表情需求

| 角色 | 需要的特殊表情 |
|------|--------------|
| 外婆 | 普通、微笑、担心、心疼、唠叨 |
| 外公 | 普通、严肃、欣慰（极少出现）、沉默 |
| 舅舅 | 普通、疲惫、皱眉、生气、苦笑、酒后 |
| 舅妈 | 普通、微笑、无奈、温柔 |
| 大表哥 | 普通、自信、无奈、认真 |
| 二表哥 | 普通、咧嘴笑、贼兮兮、无语 |
| 小明 | 普通、咧嘴笑、装大人、嘴硬、脆弱 |
| 阿旺 | 普通、面无表情、吃到好吃的亮眼睛、保护态 |
| 阿宝 | 普通、笑嘻嘻、惊恐、无辜、哭 |
| 小英 | 普通、浅笑、眼睛发亮（看到动物）、压力 |
| 小凤 | 普通、皱眉、暴怒、难得温柔、虎牙笑 |
| 主角 | 由玩家操作，表情根据状态/选择变化 |

### 美术规格建议

- 表情差分的精度高于场景中的像素小人（类似《爱氏物语》的处理）
- 每个表情是角色的半身或头部特写
- 风格保持像素感但细节更丰富（眼睛高光、面部微表情）

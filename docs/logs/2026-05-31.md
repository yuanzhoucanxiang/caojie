# 2026-05-31 工作日志

## 本次完成

### 1. InteractableObject 行动事件接口

扩展 `scripts/triggers/interactable_object.gd`，让物品除普通查看外也能作为事件节点：

- 新增 `@export` 变量：`event_id`、`event_text`、`event_conditions`、`event_choices`、`completed_description`、`repeatable`
- 条件格式沿用 NPC：`event_completed:<id>`、`attr:<属性>>=<数值>`
- 非 repeatable 事件完成后显示 `completed_description` 或普通 `description`
- 行动触发点默认 `interaction_priority = 5`

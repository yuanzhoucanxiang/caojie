# 大屋场景美术制作指南（Long Gone 风格参考）

## 参考目标

Long Gone（Hillfort Games，单人开发）画面核心特征：
- 低多边形 3D 几何体 + 像素纹理贴图
- 体积光、暖金色调、大气雾效
- 环境细节密度极高——每面墙都有故事
- 后处理：bloom、色阶、暗角、胶片颗粒

我们在 Godot 2D + gl_compatibility 下，无法做 3D。但可以通过以下组合手段追到接近的观感。

---

## 一、色调系统（最重要）

### 整体色板

Long Gone 的色板是"被太阳晒旧了的暖色世界"。不是灰暗末世，是温暖怀旧。

| 色区 | 色相 | 饱和度 | 明度 | 用途 |
|------|------|--------|------|------|
| 天空 | 暖灰黄 | 低 | 中高 | 不是蓝天，是薄雾天 |
| 远山 | 灰橄榄绿 | 低 | 中 | 远→饱和度更低（大气透视） |
| 草地 | 黄绿/卡其绿 | 中低 | 中 | 干草色，不是翠绿 |
| 红砖 | 暗赭红 | 低 | 中低 | 旧砖的褪色感 |
| 黄土 | 暖土黄 | 低 | 中 | 广东夯土墙色 |
| 木头 | 深暖棕 | 极低 | 低 | 旧木头不加高光 |
| 阴影 | 暖深棕 | 极低 | 极低 | 不要黑色，要暖棕 |

### 各场景调色参考值

```
室外院子（下午/傍晚）:
  天空:      暖灰黄     #C4B89E  (RGB约 0.77, 0.72, 0.62)
  远山:      灰橄榄     #7A8A62  (RGB约 0.48, 0.54, 0.38)
  中景草:    黄绿       #8A9E5E  (RGB约 0.54, 0.62, 0.37)
```

## 二、画面层次系统

### 当前 vs 目标

| | 当前 | Long Gone 风格目标 |
|--|------|-------------------|
| 视差层 | 5层 | 7-8层（拆细） |
| 大气雾层 | 无 | 2层（远雾+中雾） |
| 前景遮挡 | 3层(z=290/295/300) | 5层 |
| 地面纹理 | 纯色 | 带纹理+草斑+泥土变化 |

### 建议新增的层次

```
层 -2:  天空雾层（暖灰黄半透明，覆盖远景区）
        让远景山自动带上一层雾感
层 -1:  中景雾层（更淡，覆盖中景区）
层 300: 建筑前灰尘/热气扭曲（shader 效果）
层 310: 飘浮粒子（dust motes in sunlight）
```

### 大气透视怎么做

Long Gone 的远景自带"空气感"——越远越灰/越亮/越模糊。在 2D 中实现：
- 远景山的颜色本身就画得比中景更偏灰、饱和度更低
- 在远景和中景之间放一张半透明暖灰黄色块（alpha=0.08~0.12）
- 这个色块随着视差层一起滚动，模拟"远处的空气"

## 三、光影系统强化

### 阳光模拟

Long Gone 最抓眼的是光。我们能做的：
1. **顶部暖光渐变**（已有，alpha=0.12）——可加强到 alpha=0.18
2. **光束/上帝光**（新加）——几个斜向的半透明楔形色块，从画面左上向右下
3. **窗光**（室内）——一楼如果柜台旁有窗户，加一束斜光打到地面

### 阴影系统

Long Gone 里没有纯黑阴影。所有阴影都是：
- 暖深棕色
- 边缘柔软（不是硬边）
- 投影长度和角度统一（光从左上角来）

```
统一光源方向：左上→右下（太阳在画面左上方）
  建筑阴影：向右下投射，长度≈建筑高度的 0.5 倍
  角色阴影：脚下的椭圆暖深色块
  树木阴影：椭圆暖深色块，略大于树冠
  前景物阴影：投射到地面，z_index=200，半透明
```

### 室内光

| 空间 | 光源 | 色温 | 强度 |
|------|------|------|------|
| 一楼便利店 | 窗外天光+钨丝灯 | 暖黄 3000K | 中等 |
| 一楼客厅 | 窗外光+电视光 | 混合 | 偏暗 |
| 一楼厨房 | 灶膛火光+小窗 | 暖橙 2000K | 灶台周围亮，角落暗 |
| 二楼房间 | 小窗外光 | 自然光 | 较暗 |
| 三楼主角房 | 窗外天光 | 自然光偏冷 | 明亮（窗口）+暗角（角落） |

## 四、材质与纹理建议

### 像素纹理原则

Long Gone 的画面看着"丰富"不是因为分辨率高，而是因为每个表面上都有纹理变化。即使是一面白墙，也不是纯色——有污渍、裂缝、色差。

```
纯色块 → 不要用！
带纹理色块 → 同一个色块里至少有 3-4 种相近色
```

| 材质 | 纹理建议 |
|------|---------|
| 红砖墙 | 砖缝用稍深一点的暗红线条，随机几块砖颜色略不同 |
| 白灰墙 | 加点淡黄/淡灰的竖直条纹（水渍），墙角深一点 |
| 水泥地 | 随机浅灰斑点、细裂缝 |
| 泥土地 | 不同深浅的土色斑块、小草随机点缀 |
| 木桌面 | 木纹方向一致、深色节疤 |
| 瓦片屋顶 | 瓦片排列有错位，深灰+浅灰交替 |
| 灶台烟熏墙 | 从灶口向上的渐黑色，边缘模糊不规则 |

### 做法：给色块加"噪点叠加"

在 Godot 中，可以用 shader 给纯色 ColorRect 加简单纹理：
- 叠加一张无缝噪点图
- 用 `mix()` 以很低的不透明度混合
- 适合大面积的墙和地面

## 五、粒子与环境特效

Long Gone 画面里总有些小东西在动，让世界活起来：

| 特效 | 位置 | 实现方式 |
|------|------|---------|
| 阳光灰尘 | 室外光束区域 | Particles2D + 小圆点 sprite + 缓慢上飘 |
| 落叶 | 龙眼树附近 | Particles2D + 小叶 sprite + 旋转下落 |
| 炊烟 | 厨房烟囱/灶台 | Particles2D + 白灰色圆 + 螺旋上升 |
| 晾衣绳飘动 | 院子 | 衣物的 shader 做 sine wave 位移 |
| 萤火虫 | 傍晚院子 | Particles2D + 黄绿光点 + 随机浮动 |

### 粒子系统参数建议

```
阳光灰尘:
  数量: 20-30
  生命周期: 8-12s
  速度: (5, -15) ~ (15, -25)（缓慢上飘+横向漂移）
  大小: 1-3px
  颜色: 暖白略透明 alpha=0.3~0.6
  发射区域: 覆盖左上1/4画面
```

## 六、后处理强化

当前后处理 shader 参数调整：

| 参数 | 目前 | 建议 | 原因 |
|------|------|------|------|
| `vignette_intensity` | 0.25 | **0.35** | Long Gone 边缘更暗 |
| `tint_color` alpha | 0.08 | **0.12** | 整体更暖黄 |
| `tint_color` rgb | (1.0,0.92,0.82) | **(1.0,0.88,0.75)** | 更偏金色调 |

### 可新增的 shader 效果

1. **胶片颗粒** — 细微噪点叠加（alpha=0.02）
2. **色阶压缩** — 限制色彩范围，模拟胶片感
3. **微 bloom** — 亮区向外扩散 2-3px 的柔光

## 八、Long Gone 的核心设计哲学

**"不是画得细，是细节多。"**

一面墙不需要 4K 精度，但需要：
- 墙面本体的颜色变化（不是纯色）
- 墙角的水渍痕迹
- 挂着的旧日历/照片
- 脱落的墙皮下露出红砖
- 插座/开关（生活痕迹）

**"光影说了 80% 的话。"**

Long Gone 如果关了所有后期就是普通的低多边形场景。是光影（god ray、bloom、vignette、color grading）让它变成长成那样的。我们当前的 shader 系统已经有了基础，继续强化后期参数和光照层即可追上。

**"色调统一比单张画得好更重要。"**

一个场景里所有素材必须共享同一套色板。不能墙面是冷灰、地面是暖棕——那看起来像拼贴。调色时以天空颜色为基准，从上到下逐步偏暖偏暗。

# 小明妈

## 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 小明妈、亚明他妈 |
| **年龄** | 30多 |
| **身份** | 小明的母亲，独自带大小明 |
| **特点** | 沉默、坚韧、不抱怨 |

## 人物小传

小明妈是这个村子里最沉默的女人之一。

丈夫走后，她一个人扛起了整个家。种地、干家务、养孩子，日复一日。她不跟人诉苦，不跟人哭，别人提起她丈夫的事她就岔开话题。不是因为还痛——是痛过了，已经不说了。

她对小明的方式很特别：不溺爱，不打骂，但也不温柔。她让小明自己学着干活——劈柴、挑水、修东西。不是因为她狠心，是因为她知道"我不能陪他一辈子，他得学会自己照顾自己"。

她和舅妈是两种完全不同的母亲。舅妈是温暖的、话多的、把爱挂在脸上的；小明妈是冷的、话少的、把爱藏在行动里的。但两个人有一个共同点：她们都在用自己的方式撑着这个家。

村里人都敬重她。不是因为她做了什么了不起的事，是因为她什么都做了、什么都扛了、一个字都不说。

> 小明的坚强来自她。她没有给小明一个完整的家，但给了他一副硬骨头。

## 性格特点

- **沉默**：不爱说话，不是不想说，是说了也没用
- **坚韧**：什么都自己扛，不求人
- **务实**：不浪漫、不幻想，只想过好今天的日子
- **有尊严**：穷但不卑微，苦但不卖惨

## 说话方式

- 话很少，能用一个字说完的不用两个字
- 对小明说的最多的是："回来吃饭""早点睡""别给人添麻烦"
- 偶尔被小明惹生气了也只是说一句："你能不能让人省点心"
- 她不说"我爱你"，但每天的饭菜都是热的

## 与小明的关系

母子之间的爱不用语言表达。小明帮妈妈干活不说"我来帮你"，直接上手；妈妈给小明做饭不说"多吃点"，把菜往他碗里拨。

两个人之间最安静也最默契。他们不需要说的话，别人家说了一箩筐也说不完。

> 主角看到小明和妈妈的相处方式，会想起自己的爸妈——也许开始理解"不在身边不代表不爱"。

## 参与事件

- 主角去小明家、看到阿姨沉默地做饭的日常事件
- 小明帮妈妈干农活、主角默默加入的事件
- 过年时小明家安安静静、妈妈给小明做了一桌好菜的事件
- 小明妈和舅妈偶尔聊天的事件（两个女人的互相理解）

## 视觉设计参考

- 穿着：朴素的深色衣服，袖子卷着，围裙上沾着灰
- 体态：不胖不瘦，但看起来比实际年龄老一些
- 表情：大部分时候面无表情，偶尔看到小明会有一闪而过的温柔
- 动作习惯：干活很利索，走路很快，不浪费时间
- 场景：永远在干活——在地里、在厨房、在院子里

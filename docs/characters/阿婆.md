# 阿婆

## 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 阿婆 |
| **年龄** | 70+ |
| **身份** | 大屋附近的邻居老人 |
| **特点** | 爱八卦、热心、身体不太好 |

## 人物小传

阿婆是这个片区最闲的人，也是最忙的人——闲的是不用干活了，忙的是她一张嘴管着半个村的事。

她住在离外婆家不远的一间老屋里，儿女都在外面，老伴走了好几年了。她一个人住，身体不太好——腿脚不利索，走不远，天阴了膝盖就疼。但她不愿意去城里跟儿女住，说"去了就是等死，在这里好歹有人说话"。

她的日常是这样的：早上起来在门口坐着，看到路过的人就打招呼。谁家吵架了她知道，谁家来客人了她知道，谁家的鸡丢了她也知道。她不是故意打探——整个村就这么大，坐在门口什么都能听到。

她嘴碎，但不是那种讨人厌的碎。她说谁家的事不是为了嚼舌根，是因为她真的关心。她说"你舅舅又跟你舅妈吵了吧"不是看热闹，是想确认"没事吧"。她说"那个谁家的孩子又闯祸了"不是批评，是心疼。

她对外婆特别好。两个人是几十年的老邻居了，有什么好吃的都会互相送一碗。外婆忙的时候阿婆帮忙看店，阿婆生病了外婆过去照顾。她们之间不需要客套，端着碗就过去了，连门都不用敲。

对主角，阿婆是一个"永远在家"的存在。主角在村里跑来跑去，经过阿婆家门口，她永远在那儿坐着，说一句"慢点跑""饿了吧""进来喝口水"。她不参与主角的冒险，但她是主角在村里的一个锚点——不管玩到多晚、跑到多远，经过阿婆家门口就觉得快到家了。

## 性格特点

- **嘴碎**：什么都爱说两句，但没有恶意，是真的关心
- **热心**：看到谁有需要都会帮忙，虽然自己身体也不好
- **孤独**：嘴上说不孤独，但一个人坐在门口的时候，眼神会飘远
- **要强**：不愿意给儿女添麻烦，不愿意被人当"废人"看待

## 说话方式

- 语速慢，絮絮叨叨的，一件事能翻来覆去说三遍
- 喜欢拉住人说话："哎你等等""坐下坐下""我跟你说"
- 关心人的方式是唠叨："多穿点""别跑那么快""吃了没"
- 偶尔冒出一句很有智慧的话，因为她活了七十多年什么都见过

## 与外婆的关系

两个人是这个村子里最亲近的老姐妹。互相送吃的、互相照应、互相吐槽儿女。她们在一起的时候像两个小女孩——虽然都七十多了，但笑起来跟年轻时候一样。

阿婆羡慕外婆有外孙来陪。她自己的孙辈都在城里，一年见不到几次。她对主角好，有一部分是因为"看到别人家的孙子，就想起自己的"。

## 参与事件

- 日常打招呼、唠叨事件（反复出现的温馨日常）
- 主角去阿婆家喝水、吃东西的事件
- 阿婆生病了、大家担心的事件
- 阿婆讲过去的故事的事件（可能涉及外公年轻时的事）
- 阿婆的儿女打电话回来、她嘴上说"不用管我"挂了电话却坐了很久的事件

## 视觉设计参考

- 穿着：深色旧衣服，围裙，头上包着毛巾
- 体态：瘦小、驼背、走路拄着拐杖
- 表情：笑眯眯的，说话时眼睛眯成一条缝
- 动作习惯：坐在门口的竹椅上，手里可能在择菜或者摇蒲扇
- 场景：永远在门口，像一个固定的风景

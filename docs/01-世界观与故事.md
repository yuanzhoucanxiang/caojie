# 01 - 世界观与故事

## 基础设定

| 项目 | 内容 |
|------|------|
| **时间** | 约 2010 年前后 |
| **地点** | 广东省茂名市（包含县城、农村两个主舞台） |
| **语言氛围** | 粤语白话（对白可用普通话展示，辅以少量粤语口语点缀） |

---

## 故事大纲

### 序章：离开县城

主角（男孩，6岁）生活在茂名县城，父母在茂名石化上班，平时由母亲照顾。
父亲常年在外打工，母亲一个人管教不暇。主角性格调皮，但聪明。
父母商量后决定把孩子送到电白（或高州）农村，交给祖父母照顾。

### 第一幕：童年（6-12岁）

- 到达农村，水土不服、想家
- 与奶奶（祖父早逝或祖父为沉默型角色）生活
- 适应农村生活：干农活、赶集、认识村里的小朋友
- 建立与农村世界的联系
- 结束标志：小学毕业，父母面临回城读书的抉择

### 第二幕：少年（12-16岁）

- 回到县城读初中
- 城市生活节奏快、竞争激烈
- 与农村建立的联系开始疏远
- 面对新的社交圈（同学、老师）
- 结束标志：中考/某个关键选择

### 第三幕：青年（16-18岁）

- 关键转折期
- 选择决定未来方向：继续读书 / 回乡帮家人经营 / 外出打工
- 所有属性和关系最终汇合，形成不同结局

## 核心主题

| 主题 | 对应游戏机制 |
|------|-------------|
| **城乡差异** | 县城场景 vs 农村场景，不同事件池和 NPC |
| **代际关系** | 祖孙情感线，通过事件逐步加深 |
| **成长选择** | 每次选择都在塑造角色性格，无绝对好坏 |
| **地域记忆** | 年例、赶集、台风天等茂名特有的生活体验 |

---

## 茂名真实元素（游戏内呈现）

| 茂名元素 | 游戏中的表现 |
|----------|-------------|
| **年例** | 每年固定时间段的全村民俗大事件：舞狮、游神、飘色、摆宴 |
| **荔枝/龙眼种植** | 童年阶段的农活场景、家庭收入来源之一 |
| **茂名白话** | NPC 对话中偶尔插入粤语方言词，增加地域感 |
| **台风** | 某些随机事件触发：台风天关窗、收东西、停电 |
| **赶集** | 周边小镇的集市，购买生活用品、听八卦的社交场所 |
| **摩托车** | 成年后的重要交通工具 |
| **祠堂** | 村里的公共社交空间，节庆聚集地 |

| 项目 | 内容 |

## 故事大纲

### 序章：离开县城

主角（男孩，6岁）生活在茂名县城，父母去外地工作，无法兼顾照顾孩子。
父母商量后决定把孩子送到农村，交给外公外婆照顾。

### 第一幕：童年（6-12岁）

- 到达外公外婆家，面对一个陌生的大家庭
- 外公外婆家的主要收入来源：鱼塘养猪、龙眼果林
- 外公在鱼塘边有一座自己的小瓦砖房和猪圈，旁边种着芒果树，他平时独自守着鱼塘和猪，是家里的老黄牛
- 舅舅曾有自己的生意，因事业不如意回到乡下，心灰意冷，帮外公干活
- 舅妈跟着舅舅从城里来到乡下，任劳任怨扛起家务和农活，是这个家的"发动机"
- 两个表哥都不喜欢学习，早早辍学——大表哥平时在县城学厨艺找活干，周末才回来；二表哥留在乡下跟着大人
- 适应农村生活：跟表哥疯玩、帮大人干农活、赶集、认识村里其他小朋友
- 从"外来者"逐渐融入这个大家庭
- 结束标志：小学毕业，父母面临回城读书的抉择

### 第二幕：少年（12-16岁）

- 回到县城读初中
- 城市生活节奏快、竞争激烈
- 与农村建立的联系开始疏远
- 面对新的社交圈（同学、老师）
- 结束标志：中考/某个关键选择

### 第三幕：青年（16-18岁）

- 关键转折期
- 选择决定未来方向：继续读书 / 回乡帮家人经营 / 外出打工
- 所有属性和关系最终汇合，形成不同结局

## 主角家庭关系

```
外公 ─── 外婆
  │
  ├── 妈妈（主角的母亲，外出工作）
  │     └── 主角（6岁，被送来寄住）
  │
  └── 舅舅 ─── 舅妈
        ├── 大表哥（约18岁，初中辍学，在县城跟师傅学粤菜烧腊）
        └── 二表哥（约16岁，初中辍学，留在乡下）
```

| 家庭成员 | 年龄段 | 在家中的角色 | 与主角的关系 |
|---------|--------|-------------|-------------|
| **外婆** | 60+ | 操持家务、做饭、照顾全家 | 主要照顾者，情感核心 |
| **外公** | 60+ | 鱼塘和果林的主事人，话少但有威严 | 话不多，但偶尔教主角做事 |
| **舅舅** | 35-40 | 曾在城里有自己的生意，因事业不如意回到乡下，心灰意冷，现在帮外公干活 | 落寞中年人，严厉背后有疲惫 |
| **舅妈** | 35-40 | 勤劳善良的农村妇女，操持家务+养猪养鱼养果树，对两儿子辍学不强求，对主角视如己出 | 善良可亲，视如己出，不需要争取就获得的善意。和主角妈妈关系很好 |
| **大表哥** | 18 | 初中辍学，曾在外地电子厂打工，后回到茂名跟师傅学粤菜烧腊。继承舅妈的踏实肯干+舅舅的社交能力。平时在县城，周末或节假日才回乡下 | 城乡摆渡人，闯荡后逐渐理解父亲。对二表哥怒其不争但也帮他找工作 |
| **二表哥** | 16 | 初中辍学，开朗爱笑，长得像舅舅性格也像。养了一只土狗和一只狸花猫，不爱去城里（自卑），留在乡下 | 主角最亲近的玩伴，是主角"童年"本身 |

> 主角在这个家里是"外来的最小的孩子"——舅舅舅妈有自己的孩子要操心，外公外婆疼他但精力有限。这种微妙的位置感是童年阶段的核心张力之一。

## 村里人物

除了家庭成员，主角在乡下还会遇到一群同龄玩伴和村民。每个家庭都有自己的故事，表面轻松治愈，底色埋藏在玩家的探索中。

### 童年玩伴

| NPC | 全名 | 年龄 | 性格 | 对玩家的意义 |
|-----|------|------|------|-------------|
| **小明** | 亚明 | 7 | 坚强、动手能力强、讲义气。爸爸早逝，妈妈独自带大 | 主角来到村里的第一个朋友。以后会参军 |
| **阿旺** | 家旺 | 8 | 沉默、憨厚、贪吃、可靠。家境好但父母不在身边 | "不说话但在"的朋友 |
| **阿宝** | 家宝 | 7 | 话多、冒失、乐天、善良。无忧无虑但也没人管 | 快乐源泉，主角的"难兄难弟" |
| **小英** | 秀英 | 6 | 安静、细心、有压力。家人期望高 | 青梅竹马，影响"内向成长"线 |
| **小凤** | 彩凤 | 8 | 泼辣、护短、要强。家庭冷淡+重男轻女 | 青梅竹马，影响"外向成长"线 |

### 五组家庭对照

| 家庭 | 特点 | 孩子的状态 |
|------|------|-----------|
| 舅舅家 | 有温度但穷 | 被爱着但也有压力 |
| 阿旺阿宝家 | 有钱但冷清 | 物质充裕、情感空缺 |

| 小明家 | 单亲但温暖 | 坚强早熟 |

### 村里其他NPC

阿婆（邻居老人）、村长（外公的老战友）、小明妈、小凤妈，以及花婶、医生伯伯、摩托车佬等村民群像。详见 [NPC 系统](05-NPC系统.md)。

## 核心主题

| 主题 | 对应游戏机制 |
|------|-------------|
| **家庭融入** | 从寄住的"外来者"到被接纳为家庭一员 |
| **城乡差异** | 县城场景 vs 农村场景，不同事件池和 NPC |
| **代际关系** | 多代同住的大家庭，各有各的相处方式 |
| **童年友谊** | 与表哥、村里孩子的关系 |
| **成长选择** | 每次选择都在塑造角色性格，无绝对好坏 |
| **地域记忆** | 年例、赶集、台风天等茂名特有的生活体验 |

## 茂名真实元素（游戏内呈现）

| 茂名元素 | 游戏中的表现 |
|----------|-------------|
| **年例** | 每年固定时间段的全村民俗大事件：舞狮、游神、飘色、摆宴 |
| **龙眼果林** | 外公家的主要产业之一，摘龙眼、看护果林的农活事件 |
| **鱼塘** | 外公守在鱼塘边的瓦房里养鱼养猪，家里的老本行 |
| **养猪** | 外公猪圈里的土猪，喂猪和打扫猪圈的日常 |
| **芒果树** | 外公瓦房旁种的芒果树，季节限定的摘芒果事件 |
| **茂名白话** | NPC 对话中偶尔插入粤语方言词，增加地域感 |
| **台风** | 某些随机事件触发：台风天关窗、收东西、抢收鱼塘/果林 |
| **赶集** | 周边小镇的集市，购买生活用品、听八卦的社交场所 |
| **摩托车** | 成年后的重要交通工具 |
| **祠堂** | 村里的公共社交空间，节庆聚集地 |

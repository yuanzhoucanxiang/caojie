# 10 - 技术设计

## 存档系统

| 项目 | 方案 |
|------|------|
| **存档时机** | 每天睡觉时自动存档 |
| **存档内容** | 属性值、已完成事件列表、当前阶段/季节/日期、NPC关系度 |
| **存档位置** | Godot 的 `user://` 目录 |
| **格式** | JSON 文件 |
| **多存档** | 支持 3 个存档槽位 |

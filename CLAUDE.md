# caojie — 横板像素模拟经营+RPG

## 引擎
- Godot 4.6.2
- 脚本语言：GDScript

## 项目概述
玩家扮演县城出生的角色，幼时被送至乡下由祖父母照顾。通过与NPC互动和事件选择增长属性，属性影响性格和命运。
时间跨度：童年 → 少年 → 青年。

核心循环：触发事件 → 做出选择 → 属性变化 → 解锁不同后续事件/NPC反应

## 玩法
- 横版自由探索，触发事件。非动作游戏，核心是对话+选择。
- 经营元素分布在各处（干农活→体力↑，经营商店→口才/商业↑ 等）

## 美术方向
- 高精度像素美术
- 参考：Ori系列、歧路旅人(HD-2D)、REPLACED
- 2D分层素材堆叠模拟3D景深

## 项目结构
```
caojie/
├── project.godot
├── scenes/          # .tscn 场景文件
├── scripts/         # .gd 脚本文件
└── assets/
    ├── sprites/     # 角色/场景图片
    └── sounds/      # 音乐/音效
```

## 用户
- 零编程经验，做独立游戏
- 沟通语言：中文
- Godot 操作和代码需要逐步解释

## 开发策略
- 先做最小原型（1场景/1NPC/3事件），再逐步扩展
- 优先验证玩法核心，美术可以先用占位符

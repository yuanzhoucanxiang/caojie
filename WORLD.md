# caojie 游戏策划案（完整版）

> 核心概念：县城出生的孩子，幼年被送到茂名农村与外公外婆生活。通过事件选择塑造属性与性格，经历童年→少年→青年的人生轨迹。
>
> 游戏类型：2.5D 横板叙事 + 模拟经营 + 角色扮演
> 引擎：Godot 4.6.2

---

# 01 - 世界观与故事

## 基础设定

| 项目 | 内容 |
|------|------|
| **时间** | 约 2010 年前后 |
| **地点** | 广东省茂名市（包含县城、农村两个主舞台） |
| **语言氛围** | 粤语白话（对白可用普通话展示，辅以少量粤语口语点缀） |

---

## 故事大纲

### 序章：离开县城

主角（男孩，6岁）生活在茂名县城，父母去外地工作，无法兼顾照顾孩子。
父母商量后决定把孩子送到农村，交给外公外婆照顾。

### 第一幕：童年（6-12岁）

- 到达外公外婆家，面对一个陌生的大家庭
- 外公外婆家的主要收入来源：鱼塘养猪、龙眼果林
- 外公在鱼塘边有一座自己的小瓦砖房和猪圈，旁边种着芒果树，他平时独自守着鱼塘和猪，是家里的老黄牛
- 舅舅曾有自己的生意，因事业不如意回到乡下，心灰意冷，帮外公干活
- 舅妈跟着舅舅从城里来到乡下，任劳任怨扛起家务和农活，是这个家的"发动机"
- 两个表哥都不喜欢学习，早早辍学——大表哥平时在县城学厨艺找活干，周末才回来；二表哥留在乡下跟着大人
- 适应农村生活：跟表哥疯玩、帮大人干农活、赶集、认识村里其他小朋友
- 从"外来者"逐渐融入这个大家庭
- 结束标志：小学毕业，父母面临回城读书的抉择

### 第二幕：少年（12-16岁）

- 回到县城读初中
- 城市生活节奏快、竞争激烈
- 与农村建立的联系开始疏远
- 面对新的社交圈（同学、老师）
- 结束标志：中考/某个关键选择

### 第三幕：青年（16-18岁）

- 关键转折期
- 选择决定未来方向：继续读书 / 回乡帮家人经营 / 外出打工
- 所有属性和关系最终汇合，形成不同结局

## 主角家庭关系

```
外公 ─── 外婆
  │
  ├── 妈妈（主角的母亲，外出工作）
  │     └── 主角（6岁，被送来寄住）
  │
  └── 舅舅 ─── 舅妈
        ├── 大表哥（约12-13岁，高年级小学/初中）
        └── 二表哥（约8-9岁，低年级小学）
```

| 家庭成员 | 年龄段 | 在家中的角色 | 与主角的关系 |
|---------|--------|-------------|-------------|
| **外婆** | 60+ | 操持家务、做饭、照顾全家 | 主要照顾者，情感核心 |
| **外公** | 60+ | 鱼塘和果林的主事人，话少但有威严 | 话不多，但偶尔教主角做事 |
| **舅舅** | 35-40 | 曾在城里有自己的生意，因事业不如意回到乡下，心灰意冷，现在帮外公干活 | 落寞中年人，严厉背后有疲惫 |
| **舅妈** | 35-40 | 跟着舅舅从城里来到乡下，任劳任怨扛起家务和农活，是家里的"发动机" | 客气但有距离——她已经够忙了 |
| **大表哥** | 12-13 | 不喜欢读书早早辍学，在县城饭馆学厨艺，心里憋着将来开自己饭馆的野心。平时在县城，周末或节假日才回乡下，每次回来会带些县城的新鲜东西（漫画、零食）给主角，也会给家里做一顿饭 | 带头大哥，城乡之间的摆渡人，和舅舅暗中较劲 |
| **二表哥** | 8-9 | 同样不爱学习，皮，爱玩，跟着大哥混 | 主角最亲近的玩伴，会带主角疯玩 |

> 主角在这个家里是"外来的最小的孩子"——舅舅舅妈有自己的孩子要操心，外公外婆疼他但精力有限。这种微妙的位置感是童年阶段的核心张力之一。

# 02 - 角色系统

## 玩家属性

玩家角色没有"等级"概念，属性系统体现的是**性格特质**的累积。

| 属性 | 中文 | 解释 | 受哪些选择影响 |
|------|------|------|--------------|
| Obedience | 懂事 | 守规矩、体谅他人 | 听话、帮忙、忍让 |
| Curiosity | 好奇 | 探索欲、提问欲望 | 追问、闯入未知、尝试新事物 |
| Diligence | 勤劳 | 耐劳、吃苦耐劳 | 干农活、坚持学习、反复练习 |
| Stamina | 体力 | 身体素质 | 农活、运动、户外活动 |
| Closeness | 亲密 | 与祖父母的情感纽带 | 陪伴、对话选择中的亲近 |

### 属性的作用

- **解锁不同对话分支**：高好奇 → 可以追问隐藏信息；高懂事 → NPC 会更信任你
- **影响事件概率/条件**：某些事件需要属性达到门槛才触发
- **最终影响结局**：成长轨迹决定了人生的走向

### 属性范围

| 类型 | 数值范围 | 说明 |
|------|----------|------|
| 单属性 | 0 ~ 20 | 每次选择 ±1 ~ ±3 |
| 总属性和 | 动态 | 随着游戏推进不断累积 |

---

## 玩家外观成长

| 阶段 | 像素外观变化 |
|------|-------------|
| 童年（6-12岁） | 矮小、大头比例、短袖T恤+短裤 |
| 少年（12-16岁） | 长高、头身比接近正常、T恤+长裤 |
| 青年（16-18岁） | 接近成人、穿着更成熟 |

穿着变化也可通过属性间接影响：
- 回到县城后开始穿球鞋/校服
- 回农村后穿回拖鞋/短裤

---

## 成长阶段与时间推进

| 阶段 | 时长 | 触发条件 |
|------|------|----------|
| 童年 | 游戏内约 2~3 年 | 完成关键事件 "小学毕业" |
| 少年 | 游戏内约 3~4 年 | 完成关键事件 "中考" |
| 青年 | 至游戏结束 | 完成关键事件 "人生抉择" → 进入结局 |

每个阶段有若干年份，每年分为：**春季 → 夏季 → 秋季 → 冬季**，不同季节触发不同事件池。

# 03 - 核心循环

## 玩家每一轮做什么

游戏的时间单位是 **"一天"**。玩家在一天中的流程：

```
起床 → 白天活动（探索/对话/任务）→ 晚上 → 睡觉 → 进入下一天
```

### 一天的结构

| 时段 | 可做的事 |
|------|----------|
| **早晨** | 起床、和祖父母一起吃早饭（固定事件，简单对话） |
| **上午** | 自由活动：出门探索、找 NPC 对话、触发事件 |
| **下午** | 自由活动：可选择干农活/上学/做其他事 |
| **傍晚** | 回家吃饭、和家人聊几句 |
| **夜晚** | 睡觉前可看天花板沉思（属性小幅度变动）、或触发夜间事件 |

### 核心循环图

```
探索场景
    │
触发事件（对话/任务）
    │
做出选择 → 属性变化
    │
事件结束 → 解锁后续事件/关系变化
    │
继续探索 / 推进时间（到下一天或下一时段）
```

---

## 推进时间

- 事件结束后自动推进到下一个时间段
- 也可手动"回屋休息"推进时间
- 某些事件之间需要等待 N 天

# 04 - 事件系统

## 事件分类

| 类型 | 触发方式 | 示例 |
|------|----------|------|
| **剧情事件** | 必然触发，推进主线 | 初到乡下、年例、回城读书 |
| **支线事件** | 满足条件时触发 | 帮邻居修屋顶、捡到受伤小鸟 |
| **随机事件** | 一定概率每日触发 | 台风天、赶集日 |
| **日常事件** | 每天固定时段 | 吃饭、起床 |
| **年例事件** | 年例当天固定触发 | 游神、舞狮、摆宴 |
| **周末事件** | 周末/节假日触发 | 大表哥回乡下、做菜、带礼物 |

---

## 事件结构

每个事件是一个数据包：

```gdscript
{
    "id": "event_id",           # 唯一标识
    "name": "事件显示名",
    "type": "story/side/random/daily/nianli",
    "scene": "场景ID",         # 在哪个场景发生
    "conditions": [],           # 触发条件列表
    "time_window": "morning/afternoon/evening",  # 可触发时段
    "season": "summer",        # 可选，限定季节
    "dialogue": [              # 对话内容
        {
            "speaker": "NPC_ID",
            "text": "对话内容",
            "choices": [
                {
                    "text": "选项文字",
                    "effects": {"属性": +1},
                    "next": "下一组对话/事件ID"
                }
            ]
        }
    ],
    "rewards": {               # 事件完成奖励
        "attributes": {"属性": +1},
        "unlock_event": "event_id"
    }
}
```

---

## 触发条件类型

| 条件类型 | 语法 | 说明 |
|----------|------|------|
| 前置事件完成 | `event_completed:event_id` | 需要某事件已发生 |
| 属性门槛 | `attr:属性名>=值` | 需要属性达到某值 |
| 时间条件 | `day_count>=N` | 游戏天数达到某天 |
| 季节条件 | `season:summer` | 限定季节 |
| 关系条件 | `relationship:外婆>=值` | 与某 NPC 关系度 |
| 已发生次数 | `event_count:event_id>=N` | 某事件发生过 N 次 |
| 星期条件 | `weekday:weekend` | 仅周末触发（大表哥回家等） |

---

## 事件链示例：年例

```
年例前一周（春）:
├── 外婆通知 → "今年年例就在下周，你去不去看？"
│   ├── "我要去！" → 好奇+1，触发准备事件链
│   └── "不想去..." → 好奇-1，跳过准备事件
│
年例当天:
├── 起床 → 全村准备（准备事件：帮忙抬旗帜/摆供品）
├── 上午 → 游神巡游（观察事件：描述场景，可选择跟随或围观）
├── 中午 → 摆宴（社交事件：和村里孩子一起吃饭，认识新朋友）
├── 下午 → 飘色巡游（观赏事件）
├── 傍晚 → 烟花/鞭炮（自由探索时间）
└── 结束 → 外婆问"今天好玩吗？"
```

# 05 - NPC 系统

## NPC 设计原则

每个 NPC 需要定义：
- **身份**：谁、做什么的、和主角什么关系
- **性格**：典型行为模式
- **事件参与度**：TA 出现在哪些事件中
- **对话特征**：说话风格、是否有口头禅

---

### 核心 NPC —— 家庭成员

| **舅舅** | 舅舅 | 35-40 | 曾在城里做生意，事业不如意回到乡下，心灰意冷，帮外公干活，严厉但疲惫 | 落寞中年人，他的故事是童年阶段的隐藏线 |
| **舅妈** | 舅妈 | 35-40 | 跟着舅舅从城里来到乡下，任劳任怨扛起家务和农活，客气但有距离 | 家里的"发动机"，好感度需要靠主动帮忙提升 |

## NPC 关系度系统

| 关系 | 数值 | 作用 |
|------|------|------|
| 亲密值 | 0~100 | 决定对话深度和情感事件触发 |
| 信任值 | 0~100 | 决定任务委托和隐藏信息解锁 |
| 讨厌值 | 0~50 | 负面互动累积值，过高影响某些事件 |

目前原型中先只实现 **亲密值**（属性"亲密"），后续扩展。

---

## 对话系统扩展

- 事件对话：核心剧情推进
- 日常对话：每天可聊几次，内容随时间/季节变化
- 闲聊对话：NPC 的重复性对话，但有几种变体
- 默契对话：关系度高后解锁的特殊对话

## 童年阶段场景

### 1. 外婆家（主基地）

| 区域 | 内容 | 可互动元素 |
|------|------|-----------|
| 院子 | 红砖老屋+屋前空地 | 和外婆择菜、晒太阳、听故事 |
| 厨房 | 帮忙烧火、看外婆做饭 | 学做菜事件 |
| 便利店 | 一楼角落，老旧塑料罐装零食 | 挑零食、帮外婆看店、村里人来买东西闲聊 |
| 卧室 | 自己的小房间 | 睡觉、想家事件 |
| 门前 | 泥土路+村道 | 出门探索的起点 |

**美术特征**：红砖旧瓦房，木梁架，门前有水井或水龙头，土鸡在院子里走。

### 2. 村庄

| 区域 | 内容 | 可互动元素 |
|------|------|-----------|
| 村道 | 水泥路/土路，两旁是红砖房 | 遇到其他 NPC |
| 祠堂 | 村公共空间 | 年例活动、节日聚会 |
| 小卖部 | 买零食、听人闲聊 | 花零花钱、社交 |
| 竹林/榕树下 | 孩子们的玩耍聚集地 | 和小朋友玩 |

### 3. 鱼塘与果园（外公的产业）

| 区域 | 内容 | 可互动元素 |

# 人物档案

## 外婆

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 外婆 |
| **年龄** | 60+ |
| **身份** | 外祖母，家里的精神支柱 |

### 人物小传

外婆是这个家的根。她一辈子没离开过这片土地，外公、舅舅、两个孙子，再加上一个从县城送来的小外孙，全部靠她一个人的耐心撑着。

她老了，重活已经干不动了，但一日三餐从来没落下过。灶台是她的战场，厨房是她说了算的地方。她做饭不精致，但量大、管饱、有家的味道。

大屋一楼有一角是她开的小便利店，用一些老旧的塑料罐子装着零食来卖——瓜子、花生、糖果、辣条，还有小孩爱吃的零碎东西。赚不了几个钱，但她觉得闲着也是闲着，有人来买还能顺便聊几句。罐子是攒了很多年的，有的发黄了，有的盖子裂了用橡皮筋箍着。主角有时候会蹲在罐子前面挑零食，外婆就在旁边看着，偶尔说"这个辣，你吃不了"。

她对主角是真心疼的。这个孩子不是自己生的，但来了就是自己家的。她会偷偷给主角塞零食，会在其他大人不在的时候多说几句贴心话。但她的精力有限，要操心的事情太多，有时候顾不上主角的情绪——不是不关心，是真的忙不过来。

### 性格特点

- **温和絮叨**：嘴上总在说"吃饱了没有""多穿点""别跑太快"，是那种用唠叨表达爱的人
- **传统**：信老规矩，逢年过节该拜的拜、该做的做，年例的时候她比谁都认真
- **有耐心**：对几个孩子都很包容，不打不骂，最多叹口气
- **隐忍**：家里的难处她都知道——舅舅的心灰、舅妈的辛苦、外公的沉默——但她不说破，默默撑着

### 说话方式

- 语速慢，带着茂名白话的腔调
- 口头禅："乖孙""食咗饭未啊"（吃了没有）
- 对主角的称呼随心情变：叫名字、叫"乖孙"、叫"细佬仔"（小孩子）
- 偶尔冒出一两句老话、谚语

### 与女儿（主角妈妈）的感情

女儿嫁出去了，去了外地，老公也得出去打工，日子过得不容易，孩子没人带，只能送回来。

外婆心疼女儿，但心里也有一个说不出口的想法：你小时候我带你，老了还要帮你带你的孩子。但她说不出来。她不会跟女儿抱怨，也不会跟主角说"你妈不要你了"这种话。她就是接着，默默接着。

逢年过节女儿打电话回来，她会说"孩子好着呢，你放心"，然后挂了电话叹口气。

她对主角的好，有一部分是对女儿的牵挂——你照顾好她的孩子，就是帮了她最大的忙。主角是外婆和女儿之间的情感纽带，外婆照顾主角，某种程度上也是在替女儿尽母亲的责任。

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 主角刚到，对外婆有距离感，想家 |
| **发展** | 日常相处积累亲密感，择菜、做饭、赶集 |
| **关键转折** | 某个主角特别想家/受委屈的夜晚，外婆的安慰 |

> 外婆是主角在乡下最先建立情感联系的人。她不会说什么大道理，但她的存在本身就是一种安全感。

### 参与事件

- 择菜、做饭、赶集（日常活动）
- 年例准备（拜神、备菜）
- 台风天（关窗、收东西）
- 主角想家时的安慰事件

### 视觉设计参考

- 穿着：深色碎花布衫、黑布裤、塑料拖鞋
- 体态：微驼背、走不快、手上有老茧
- 表情：大部分时候是慈祥的笑，偶尔叹气时眉头微皱

## 外公

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 外公 |
| **年龄** | 60+ |
| **身份** | 外祖父，鱼塘和猪圈的守塘人 |

### 人物小传

外公是这个家最沉默的人。

他在鱼塘边有一座自己的小瓦砖房，旁边是猪圈，房前种了几棵芒果树。他不像其他人住在外婆家的大屋里，而是长年一个人守在这片产业上。喂鱼、养猪、巡塘、修补——日复一日。

他是这个家真正的老黄牛。鱼塘和猪是家里主要收入来源之一，外公一个人撑着这块。他不抱怨，也不多说话，但如果你仔细看，他的手比谁的都粗糙，脸比谁的都黑。

他的沉默不是天生的。年轻的时候，他被卷入过一场南边的仗。他不是正式的军人，但那场战争还是在他身上留下了痕迹。从那以后，他就变成了一个不爱说话的人。他不再跟人解释什么，也不再对事情发表看法——做就是了，别问为什么。

他对主角不会主动亲近。他不哄孩子，也不逗孩子。但如果你去鱼塘找他，他会默默教你怎么做——怎么撒饲料、怎么看鱼浮上来的样子、怎么判断天会不会下雨。他不说"我教你"，他直接做，你看不看、学不学，是你的事。

他住在鱼塘边不是因为跟外婆有什么矛盾，而是他觉得住在这里最方便看塘。鱼塘是家里的饭碗，他守着饭碗，天经地义。鱼塘离大屋很近，他早上有时候会回大屋一趟——手里提着一桶小鱼，放到主角面前，让主角自己挑几条去养。他不说这是给你的，他就是把桶放下。芒果熟了也会摘几个带回来，往桌上一放就走。

这就是他的爱的方式——不解释，不邀功，只是默默把东西带到你面前。

### 性格特点

- **沉默寡言**：一天说的话可能不超过十句，能用动作表达就不开口
- **实干**：不讲虚的，鱼塘的事、猪圈的事，他亲力亲为
- **有威严**：不发火，但家里人都怕他。舅舅在外面算个生意人，回来在老爹面前一样老实
- **独立**：不住大屋，自己守着鱼塘瓦房，不给任何人添麻烦
- **手艺好**：修补鱼塘、搭猪圈、砍芒果树枝，什么都会做
- **被战争改变过**：年轻时卷入过一场南边的仗（非正式军人），回来后性格大变，从那以后不再多说话

### 年轻时的外公

外公年轻的时候跟现在判若两人。他被卷入过一场南边的仗，不是作为正式的军人，但那场战争在他身上留下了不可逆的痕迹。具体经历了什么，他从来没跟任何人说过——包括外婆。

从那以后，他不再跟人解释什么，不再对事情发表看法。他把所有精力投到了鱼塘和猪圈上——也许是因为只有这些实实在在的活，才能让他不去想那些事。

> **游戏内模糊化表达**：不直接提及"对越自卫反击战"，用"南边的仗""那年的事""从战场回来以后"等模糊表述。玩家不需要知道具体是什么战争，只需要感受到一个被改变过的老人。

### 说话方式

- 话极少，经常只说半句："去那边""看着""嗯"
- 不说教，不讲道理，用行动示范
- 对主角偶尔蹦出一句让人记住很久的话，比如"鱼要慢慢喂，急不得"

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 主角觉得外公很可怕，不敢靠近 |
| **发展** | 主角主动去鱼塘找外公，外公默默接受这个"小跟班" |
| **关键转折** | 外公第一次主动教主角做事（可能是喂鱼），没有解释但用行动示范 |

> 外公是主角"沉默中的关爱"的来源。他不表达，但如果你感受得到，那就是最深的一种信任。

### 与舅舅的关系

外公和舅舅是这个家里最沉默的一对父子。

舅舅在城里做过生意，见过世面，性格比外公话多、主意多、脾气也更外放。但回到乡下后，他在老爹面前收敛了。外公不发火，也不评价舅舅城里的失败，从头到尾一个字不提。但这种沉默比任何责骂都让舅舅难受——沉默可以被解读为"我早就知道你会这样"。

舅舅偶尔会冒出一些"城里怎样怎样"的话，或者对鱼塘的做法有自己的想法，但外公不接茬，继续按自己的方式干。舅舅觉得老爹听不进去，外公觉得你连自己都顾不好还想教我。两人不吵架，但暗涌不断。

不过真遇到大事——台风来了、鱼塘出问题——舅舅是能扛的，外公也知道。父子俩平时各干各的，危机时刻不用说就站到一起了。

> 外公是"做不说"的人，舅舅是"说但没做成"的人。方向相反，但根上是一样的倔。

### 参与事件

- 喂鱼、捞鱼（鱼塘活动）
- 喂猪、打扫猪圈（养猪活动）
- 去小瓦房找外公（亲密事件）
- 早上带回一桶小鱼让主角挑（日常温情事件，反复触发）
- 芒果季带芒果回来（季节事件）
- 暴雨天抢收鱼塘（危机事件）

### 视觉设计参考

- 穿着：旧军绿解放鞋、灰蓝布衫、宽腿裤
- 体态：精瘦结实、皮肤黝黑、手上有厚厚的老茧
- 表情：大部分时候面无表情，偶尔嘴角微动算是笑了

## 舅舅

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 舅舅 |
| **年龄** | 35-40 |
| **身份** | 舅舅，曾在城里做生意，现在帮外公干农活 |

### 人物小传

舅舅是这个家最复杂的人。

他曾经在茂名市区有自己的生意，是一个见过世面的人。但事业上遭遇了不如意——具体什么原因可以后续展开——他带着舅妈回到了乡下。

回到乡下后，他帮外公打理鱼塘和果林。活是干的，但心是灰的。他不像外公那样心无旁骛地守着产业，他干着活的时候可能在想别的事——那些没做成的生意、没赚到的钱、没实现的野心。

他在家里是"不上不下"的位置。对外公他是儿子，要听话；对舅妈他有愧疚——人家跟着自己从城里来到这乡下地方；对两个儿子他可能不知道怎么面对——自己都活不明白，拿什么教育孩子。

但他不是坏人。他严厉，是因为他对自己也严厉。他不善于表达关心，是因为没人教过他怎么表达。如果家里遇到大事——比如台风、比如需要做重要决定——他是会站出来的那个人。

### 性格特点

- **严厉**：对孩子们要求高，但不会好好说，总是用批评的语气
- **疲惫**：不是身体累，是心累。从城里退回来的中年人都有这种疲惫
- **不善表达**：关心人的方式是默默干活、把事情做好，而不是说好听的话
- **有担当**：平时看着消沉，真遇到事他是能扛的
- **敏感**：别人一句无心的话可能戳到他的痛处，尤其是关于"城里"的话题

### 说话方式

- 语气偏硬，不是吵架的硬，是不知道怎么柔软下来的那种硬
- 不怎么说过去的事，关于城里的生意只字不提
- 对主角说话简短直接："去帮忙""别乱跑""吃饭了"
- 喝了酒之后可能会多说几句，但说的也都是牢骚

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 舅舅对主角客气但疏远，多一张嘴吃饭而已 |
| **发展** | 主角跟在舅舅后面干活，舅舅偶尔教一两句，态度慢慢软化 |
| **关键转折** | 某次舅舅喝了酒或者遇到什么事，对主角说了一些真心话 |

> 舅舅是主角理解"大人也不容易"的窗口。他不是英雄，也不是反派，只是一个被生活打败了但还在硬撑的人。

### 与外公的关系

舅舅和外公是这个家里最沉默的一对父子。

外公是"做不说"的人，一辈子不解释、不评价。舅舅是"说但没做成"的人——在城里做过生意，见过世面，性格比外公话多、主意多、脾气也更外放。但回到乡下后，他在老爹面前收敛了。

外公从头到尾不评价舅舅城里的失败，一个字不提。但这种沉默比任何责骂都让舅舅难受——沉默可以被解读为"我早就知道你会这样"。舅舅偶尔会冒出一些"城里怎样怎样"的话，或者对鱼塘的做法有自己的想法，但外公不接茬，继续按自己的方式干。舅舅觉得老爹听不进去，外公觉得你连自己都顾不好还想教我。

两人不吵架，但暗涌不断。真遇到大事——台风来了、鱼塘出问题——舅舅是能扛的，外公也知道。父子俩平时各干各的，危机时刻不用说就站到一起了。

### 与大表哥的暗线

舅舅从城里退回来，大表哥往城里闯——父子俩方向相反。舅舅可能觉得大表哥迟早也会撞南墙，大表哥觉得舅舅是被打败了才回来。他们不吵架，但互相不理解。这种沉默的对峙比激烈的冲突更有力量。

### 参与事件

- 干农活、果林劳作（日常活动）
- 暴雨天抢收（危机事件）
- 年例期间的组织帮忙
- 酒后吐真言事件（情感事件）

### 视觉设计参考

- 穿着：不太讲究，旧T恤+宽裤+人字拖
- 体态：中等身材，干活练出来的肌肉，但脸上的疲惫藏不住
- 表情：大部分时候是沉着脸，偶尔苦笑

## 舅妈

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 舅妈 |
| **年龄** | 35-40 |
| **身份** | 舅舅的妻子，从城里跟来乡下，家里的"发动机" |

### 人物小传

舅妈是这个家最不容易被看见的人，但没有她这个家转不动。

她跟着舅舅从城里来到乡下。别人可能觉得她是"嫁鸡随鸡"，但只有她自己知道这个决定意味着什么——离开城市的生活，来到一个她并不熟悉的地方，照顾两个老人、两个自己的孩子、还有一个不是自己生的小外孙。

她不抱怨。不是不想抱怨，是抱怨没用。外婆年纪大了操持不动重活，舅舅心灰意冷顾不上家务，外公守在鱼塘不回来，大表哥在县城学手艺，二表哥还是个皮孩子。能干活的人不多，她不干谁干。

她对主角的态度是"客气但有距离"。不是冷，是真的没有多余的精力去热情。她有自己的两个孩子要操心，多一个外孙，能保证他吃饱穿暖、不饿着冻着，已经尽了她的本分。但如果主角主动帮她干活——择菜、扫地、喂鸡——她会慢慢软下来。

### 性格特点

- **勤劳**：从早到晚不停，家务农活两头跑
- **精明**：过日子会算计，不浪费一分钱、一粒米
- **务实**：不讲虚的，也不说漂亮话，做事就是做事
- **有距离感**：不是冷漠，是真的忙、真的累，顾不上热情
- **嘴硬心软**：嘴上说"这孩子怎么这样"，但该照顾的一样不少

### 说话方式

- 语速快，干活的时候边干边说
- 不说废话，吩咐事情干脆利落："去把鸡喂了""菜端上去"
- 偶尔对主角说一两句关心的话，但说完就走，不给你回应的时间
- 对舅舅有时会抱怨两句，但适可而止

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 客气，保持距离，按基本标准照顾 |
| **发展** | 主角主动帮忙干活，舅妈开始多说几句话、多留意主角 |
| **关键转折** | 某次主角做了什么让舅妈意外的事（可能是帮了大忙），舅妈第一次真正认可这个"外来的孩子" |

> 舅妈是主角理解"被接纳需要努力"的NPC。她不欠主角什么，但她的态度变化是最有分量的——因为她不是一开始就对你好的人。

### 参与事件

- 家务活动（择菜、扫地、喂鸡）
- 赶集（和外婆一起去）
- 年例备菜
- 某次舅妈生病/累倒，主角帮忙撑了一天的家务（关键事件）

### 视觉设计参考

- 穿着：朴素但利落，干活时袖子永远卷到手肘
- 体态：结实、不胖，操劳的人没有胖的
- 表情：大部分时候是"忙着呢别烦我"的表情，偶尔笑的时候眼角有皱纹

## 大表哥

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 大表哥 |
| **年龄** | 12-13 |
| **身份** | 舅舅的大儿子，在县城学厨艺，周末/节假日回乡下 |

### 人物小传

大表哥是这个家最"往外走"的人。

他不爱读书，早早辍了学。但他没有像舅舅那样"退"回来，而是主动往县城跑——在饭馆里找活干、学厨艺。他心里憋着一个没跟家里人说过的野心：将来要在茂名开一间自己的大排档或小饭馆。

他为什么不说？因为他见过舅舅是怎么被"城里"打败的。如果他说"我要去城里闯"，家里人（尤其是舅舅）会怎么想？所以他只做不说，在县城默默学手艺，周末回来给家里做一顿饭。他的手艺在慢慢进步——第一次做的菜可能不怎么样，但每一次回来都比上一次好一点。

他是主角认识"外面世界"的窗口。每次回来，他会给主角带点县城的东西——一包零食、一本漫画、一个小玩意。这些东西在乡下买不到，对主角来说就是"外面的证据"。他也会跟主角讲县城的见闻，但不是炫耀，而是分享——他把主角当一个可以聊天的人。

他和舅舅之间有一条暗线。舅舅从城里退回来，他从乡下往城里闯。父子俩方向相反，不吵架，但互相不理解。大表哥觉得舅舅是被打败了才回来的，舅舅觉得大表哥迟早也会撞南墙。他们用沉默对峙。

### 性格特点

- **有主见**：知道自己要什么，不随大流
- **懂事**：比同龄人成熟，可能是家庭变故逼出来的
- **有野心但低调**：想开饭馆但不张扬，只在跟主角聊天时偶尔提一句
- **仗义**：对主角是真的好，不是敷衍，而是当弟弟看
- **倔**：跟舅舅一样倔，不肯服软

### 说话方式

- 在主角面前话多，愿意分享、愿意教
- 在大人面前话少，尤其是舅舅面前
- 口头禅："没啥大不了的""等我以后……"
- 偶尔带点县城的词或者饭馆里的行话

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 大表哥不常在家，主角对他不熟，只有周末才见到 |
| **发展** | 大表哥每次回来给主角带东西、讲故事，关系逐渐亲近 |
| **关键转折** | 大表哥对主角说了自己想开饭馆的秘密——这是他第一次对人说出来 |

> 大表哥是主角的"榜样型玩伴"——他比主角大，见过更多东西，有自己想做的事。他让主角看到"不读书也有出路"，但也让主角思考"那条路是不是我想要的"。

### 关键弧线

大表哥的做菜手艺是可视化的成长线：
1. 第一次回来做的菜，味道一般，但全家人都给了面子
2. 中间几次，有进步但也有翻车（比如盐放多了）
3. 某一次做了一道真正好吃的菜，全家人都惊讶
4. 最后阶段，他做的菜已经像模像样，甚至外婆都点头认可

这个过程不需要玩家做什么，只需要"在场"——看着他变化。

### 参与事件

- 周末回家做菜（定期事件）
- 给主角带礼物（触发好奇心属性增长）
- 讲县城见闻（触发懂事属性增长）
- 和舅舅的沉默对峙（家庭暗线事件）
- 说出口想开饭馆（情感事件）

### 视觉设计参考

- 穿着：比家里其他人稍微讲究一点，偶尔穿件县城买的T恤
- 体态：瘦但结实，干活练出来的
- 表情：在家时放松的笑，在舅舅面前偏严肃
- 随身物品：偶尔带个小包回来，里面装着给主角的东西

## 二表哥

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 二表哥 |
| **年龄** | 8-9 |
| **身份** | 舅舅的小儿子，留在乡下，主角最亲近的玩伴 |

### 人物小传

二表哥是这个家最快乐的人。

他跟大哥一样不爱读书，早早辍了学。但跟大哥不同的是，他完全没有"我要学手艺""我要去城里"这种想法。他的世界就是这个村子——鱼塘边的泥巴、果林里的鸟窝、村口的竹林、河沟里的蝌蚪。

他是主角来到乡下后第一个真正玩到一起的人。大表哥不常在家，大人们各有各的事，只有二表哥是随时可以叫出来疯的。他会带主角去各种主角不敢去的地方——钻进果林深处、跑到鱼塘边抓青蛙、翻过田埂去看邻村的孩子放牛。

他皮，但不是熊孩子那种皮。他不做坏事，只是精力旺盛、坐不住。他对主角是真心仗义的——谁欺负主角他第一个冲上去，有好吃的会分一半给主角。

他是这个家最不"懂事"的人，但也正因为不"懂事"，他活得最轻松。大人们的心事、家里的经济、舅舅的失意——这些他都不管。他的世界很简单：今天玩什么、明天去哪里玩。

### 性格特点

- **皮**：坐不住、跑不停、永远有使不完的劲
- **仗义**：对朋友好，有东西分一半，有人欺负你他帮你出头
- **好奇心强**：什么都想看、什么都想摸、什么都想试
- **天不怕地不怕**：不害怕虫子、不害怕黑、不害怕外公的黑脸
- **大大咧咧**：被骂了转头就忘，不记仇、不纠结

### 说话方式

- 话多且快，像机关枪一样
- 口头禅："走！""你来不来！""这个好玩！"
- 经常叫主角的外号（可能自己给主角取了一个）
- 偶尔冒出一句让人哭笑不得的话

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 二表哥主动来找主角玩，主角还扭扭捏捏 |
| **发展** | 一起疯、一起野、一起闯祸，变成铁哥们 |
| **关键转折** | 某次一起闯了祸，二表哥主动替主角背锅 |

> 二表哥是主角"童年"本身。跟大表哥在一起是"学到东西"，跟二表哥在一起是"玩得开心"。当主角将来面临回城读书的选择时，二表哥是那个最不舍的人。

### 参与事件

- 日常一起玩（探索村庄、鱼塘、果林）
- 一起闯祸事件（搞笑为主）
- 年例一起看舞狮、放鞭炮
- 主角要回城读书时的分别事件

### 视觉设计参考

- 穿着：脏兮兮的，膝盖上永远有泥巴，鞋子经常穿反
- 体态：比同龄人结实，到处跑练出来的
- 表情：咧嘴笑、眼睛亮亮的、做坏事时贼兮兮的
- 随身物品：口袋里可能装着弹珠、石子、抓到的小虫

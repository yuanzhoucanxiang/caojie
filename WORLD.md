# caojie 游戏策划案（完整版）

> 核心概念：县城出生的孩子，幼年被送到茂名农村与外公外婆生活。通过事件选择塑造属性与性格，经历童年→少年→青年的人生轨迹。
>
> 游戏类型：2.5D 横板叙事 + 模拟经营 + 角色扮演
> 引擎：Godot 4.6.2

---

# 01 - 世界观与故事

## 基础设定

| 项目 | 内容 |
|------|------|
| **时间** | 约 2010 年前后 |
| **地点** | 广东省茂名市（包含县城、农村两个主舞台） |
| **语言氛围** | 粤语白话（对白可用普通话展示，辅以少量粤语口语点缀） |

## 故事大纲

### 序章：离开县城

主角（男孩，6岁）生活在茂名县城，父母去外地工作，无法兼顾照顾孩子。
父母商量后决定把孩子送到农村，交给外公外婆照顾。

### 第一幕：童年（6-12岁）

- 到达外公外婆家，面对一个陌生的大家庭
- 外公外婆家的主要收入来源：鱼塘养猪、龙眼果林
- 外公在鱼塘边有一座自己的小瓦砖房和猪圈，旁边种着芒果树，他平时独自守着鱼塘和猪，是家里的老黄牛
- 舅舅曾有自己的生意，因事业不如意回到乡下，心灰意冷，帮外公干活
- 舅妈跟着舅舅从城里来到乡下，任劳任怨扛起家务和农活，是这个家的"发动机"
- 两个表哥都不喜欢学习，早早辍学——大表哥平时在县城学厨艺找活干，周末才回来；二表哥留在乡下跟着大人
- 适应农村生活：跟表哥疯玩、帮大人干农活、赶集、认识村里其他小朋友
- 从"外来者"逐渐融入这个大家庭
- 结束标志：小学毕业，父母面临回城读书的抉择

### 第二幕：少年（12-16岁）

- 回到县城读初中
- 城市生活节奏快、竞争激烈
- 与农村建立的联系开始疏远
- 面对新的社交圈（同学、老师）
- 结束标志：中考/某个关键选择

### 第三幕：青年（16-18岁）

- 关键转折期
- 选择决定未来方向：继续读书 / 回乡帮家人经营 / 外出打工
- 所有属性和关系最终汇合，形成不同结局

## 主角家庭关系

```
外公 ─── 外婆
  │
  ├── 妈妈（主角的母亲，外出工作）
  │     └── 主角（6岁，被送来寄住）
  │
  └── 舅舅 ─── 舅妈
        ├── 大表哥（约18岁，初中辍学，在县城学厨艺）
        └── 二表哥（约16岁，初中辍学，留在乡下）
```

| 家庭成员 | 年龄段 | 在家中的角色 | 与主角的关系 |
|---------|--------|-------------|-------------|
| **外婆** | 60+ | 操持家务、做饭、照顾全家 | 主要照顾者，情感核心 |
| **外公** | 60+ | 鱼塘和果林的主事人，话少但有威严 | 话不多，但偶尔教主角做事 |
| **舅舅** | 35-40 | 曾在城里有自己的生意，因事业不如意回到乡下，心灰意冷，现在帮外公干活 | 落寞中年人，严厉背后有疲惫 |
| **舅妈** | 35-40 | 勤劳善良的农村妇女，操持家务+养猪养鱼养果树，对两儿子辍学不强求，对主角视如己出 | 善良可亲，视如己出，不需要争取就获得的善意。和主角妈妈关系很好 |
| **大表哥** | 18 | 初中辍学，曾在外地电子厂打工，后回到茂名跟师傅学粤菜烧腊。继承舅妈的踏实肯干+舅舅的社交能力强。平时在县城，周末或节假日才回乡下 | 城乡摆渡人，闯荡后逐渐理解父亲。对二表哥怒其不争但也帮他找工作 |
| **二表哥** | 16 | 初中辍学，开朗爱笑，长得像舅舅性格也像。养了一只土狗和一只狸花猫，不爱去城里（自卑），留在乡下 | 主角最亲近的玩伴，是主角"童年"本身 |

> 主角在这个家里是"外来的最小的孩子"——舅舅舅妈有自己的孩子要操心，外公外婆疼他但精力有限。这种微妙的位置感是童年阶段的核心张力之一。

# 02 - 角色系统

## 玩家属性

玩家角色没有"等级"概念，属性系统体现的是**性格特质**的累积。

| 属性 | 中文 | 解释 | 受哪些选择影响 |
|------|------|------|--------------|
| Obedience | 懂事 | 守规矩、体谅他人 | 听话、帮忙、忍让 |
| Curiosity | 好奇 | 探索欲、提问欲望 | 追问、闯入未知、尝试新事物 |
| Diligence | 勤劳 | 耐劳、吃苦耐劳 | 干农活、坚持学习、反复练习 |
| Stamina | 体力 | 身体素质 | 农活、运动、户外活动 |
| Closeness | 亲密 | 与祖父母的情感纽带 | 陪伴、对话选择中的亲近 |

### 属性的作用

- **解锁不同对话分支**：高好奇 → 可以追问隐藏信息；高懂事 → NPC 会更信任你
- **影响事件概率/条件**：某些事件需要属性达到门槛才触发
- **最终影响结局**：成长轨迹决定了人生的走向

### 属性范围

| 类型 | 数值范围 | 说明 |
|------|----------|------|
| 单属性 | 0 ~ 20 | 每次选择 ±1 ~ ±3 |
| 总属性和 | 动态 | 随着游戏推进不断累积 |

---

## 玩家外观成长

| 阶段 | 像素外观变化 |
|------|-------------|
| 童年（6-12岁） | 矮小、大头比例、短袖T恤+短裤 |
| 少年（12-16岁） | 长高、头身比接近正常、T恤+长裤 |
| 青年（16-18岁） | 接近成人、穿着更成熟 |

穿着变化也可通过属性间接影响：
- 回到县城后开始穿球鞋/校服
- 回农村后穿回拖鞋/短裤

---

## 成长阶段与时间推进

| 阶段 | 时长 | 触发条件 |
|------|------|----------|
| 童年 | 游戏内约 2~3 年 | 完成关键事件 "小学毕业" |
| 少年 | 游戏内约 3~4 年 | 完成关键事件 "中考" |
| 青年 | 至游戏结束 | 完成关键事件 "人生抉择" → 进入结局 |

每个阶段有若干年份，每年分为：**春季 → 夏季 → 秋季 → 冬季**，不同季节触发不同事件池。

# 03 - 核心循环

## 玩家每一轮做什么

游戏的时间单位是 **"一天"**。玩家在一天中的流程：

```
起床 → 白天活动（探索/对话/任务）→ 晚上 → 睡觉 → 进入下一天
```

### 一天的结构

| 时段 | 可做的事 |
|------|----------|
| **早晨** | 起床、和祖父母一起吃早饭（固定事件，简单对话） |
| **上午** | 自由活动：出门探索、找 NPC 对话、触发事件 |
| **下午** | 自由活动：可选择干农活/上学/做其他事 |
| **傍晚** | 回家吃饭、和家人聊几句 |
| **夜晚** | 睡觉前可看天花板沉思（属性小幅度变动）、或触发夜间事件 |

### 核心循环图

```
探索场景
    │
触发事件（对话/任务）
    │
做出选择 → 属性变化
    │
事件结束 → 解锁后续事件/关系变化
    │
继续探索 / 推进时间（到下一天或下一时段）
```

---

## 推进时间

- 事件结束后自动推进到下一个时间段
- 也可手动"回屋休息"推进时间
- 某些事件之间需要等待 N 天

# 04 - 事件系统

## 事件分类

| 类型 | 触发方式 | 示例 |
|------|----------|------|
| **剧情事件** | 必然触发，推进主线 | 初到乡下、年例、回城读书 |
| **支线事件** | 满足条件时触发 | 帮邻居修屋顶、捡到受伤小鸟 |
| **随机事件** | 一定概率每日触发 | 台风天、赶集日 |
| **日常事件** | 每天固定时段 | 吃饭、起床 |
| **年例事件** | 年例当天固定触发 | 游神、舞狮、摆宴 |
| **周末事件** | 周末/节假日触发 | 大表哥回乡下、做菜、带礼物 |

---

## 事件结构

每个事件是一个数据包：

```gdscript
{
    "id": "event_id",           # 唯一标识
    "name": "事件显示名",
    "type": "story/side/random/daily/nianli",
    "scene": "场景ID",         # 在哪个场景发生
    "conditions": [],           # 触发条件列表
    "time_window": "morning/afternoon/evening",  # 可触发时段
    "season": "summer",        # 可选，限定季节
    "dialogue": [              # 对话内容
        {
            "speaker": "NPC_ID",
            "text": "对话内容",
            "choices": [
                {
                    "text": "选项文字",
                    "effects": {"属性": +1},
                    "next": "下一组对话/事件ID"
                }
            ]
        }
    ],
    "rewards": {               # 事件完成奖励
        "attributes": {"属性": +1},
        "unlock_event": "event_id"
    }
}
```

---

## 触发条件类型

| 条件类型 | 语法 | 说明 |
|----------|------|------|
| 前置事件完成 | `event_completed:event_id` | 需要某事件已发生 |
| 属性门槛 | `attr:属性名>=值` | 需要属性达到某值 |
| 时间条件 | `day_count>=N` | 游戏天数达到某天 |
| 季节条件 | `season:summer` | 限定季节 |
| 关系条件 | `relationship:外婆>=值` | 与某 NPC 关系度 |
| 已发生次数 | `event_count:event_id>=N` | 某事件发生过 N 次 |
| 星期条件 | `weekday:weekend` | 仅周末触发（大表哥回家等） |

---

## 事件链示例：年例

```
年例前一周（春）:
├── 外婆通知 → "今年年例就在下周，你去不去看？"
│   ├── "我要去！" → 好奇+1，触发准备事件链
│   └── "不想去..." → 好奇-1，跳过准备事件
│
年例当天:
├── 起床 → 全村准备（准备事件：帮忙抬旗帜/摆供品）
├── 上午 → 游神巡游（观察事件：描述场景，可选择跟随或围观）
├── 中午 → 摆宴（社交事件：和村里孩子一起吃饭，认识新朋友）
├── 下午 → 飘色巡游（观赏事件）
├── 傍晚 → 烟花/鞭炮（自由探索时间）
└── 结束 → 外婆问"今天好玩吗？"
```

# 05 - NPC 系统

## NPC 设计原则

每个 NPC 需要定义：
- **身份**：谁、做什么的、和主角什么关系
- **性格**：典型行为模式
- **事件参与度**：TA 出现在哪些事件中
- **对话特征**：说话风格、是否有口头禅

## 青梅竹马好感度系统

小英和小凤是主角的两位青梅竹马，好感度贯穿童年→少年→青年三个阶段。

| 角色 | 性格类型 | 好感度提升方向 | 故事影响 |
|------|---------|--------------|---------|
| **小英** | 安静内敛 | 安静的陪伴、细腻的关心、一起看小动物 | 影响"内向成长"线——主角学会安静、观察、共情 |

## 06 - 场景与世界

### 画面布局

#### 场景分层（从远到近）

| 层 | 内容 | 视差速度 | 说明 |
|----|------|---------|------|
| 天空层 | 天空、云、太阳/月亮 | 最慢 | 随时间段变色（白天蓝→傍晚橙→夜晚深蓝） |
| 远景层 | 远处的山、树林轮廓 | 很慢 | 剪影式处理，不需要细节 |
| 中景层 | 村子的远景、其他人家屋顶 | 慢 | 能看到远处邻居的红砖房、竹林 |
| 场景层 | 主要游戏区域 | 跟随玩家 | 大屋、院子、村道等所有可互动内容 |
| 前景层 | 近处遮挡物 | 比玩家快 | 树枝垂下来、草丛、龙眼树叶 |

玩家在场景层上，可前后移动（深度缩放），靠近前景时角色变大、远离时变小。

#### 室外 vs 室内视角

室外和室内使用不同的镜头距离，参考《直到那时》的室内拉近处理。

| | 室外 | 室内 |
|--|------|------|
| 镜头距离 | 较远，能看全景 | 较近，聚焦角色 |
| 画面层次 | 天空+远景+场景+前景，多层视差 | 墙壁+家具+人物，层次简化 |
| 感受 | 开阔、自由、可以去任何地方 | 亲密、安静、私密 |
| 参考 | Ori 的自然场景 | 《直到那时》的房间/走廊 |

**外婆大屋内的镜头运用：**

| 区域 | 镜头处理 |
|------|---------|
| 院子 | 室外视角，较远，能看到天空和周围环境 |
| 一楼客厅/厨房 | 室内视角，拉近，能看到灶台、圆桌、墙上的挂历 |
| 二楼走廊 | 室内视角，窄画面，走廊感 |
| 主角小房间 | 最近的镜头，房间很小，聚焦在床上和窗外 |
| 天台 | 室外视角但更开阔——从拉近的室内切到天台，有呼吸感 |

室内→室外的镜头切换本身也是一种叙事：从拉近的"家"走到拉远的"外面世界"，玩家直观感受到空间的变化。天台是特殊的过渡空间——物理上在大屋里，视觉上是室外。

#### 摄像机

- 水平跟随玩家，平滑滚动
- 场景有左右边界，到达边界时触发场景切换
- 室内外切换时镜头有平滑过渡动画

#### 场景切换

场景切换使用**像素化水彩过渡**效果——水彩的晕开感但保持像素网格，和漫画框对话风格统一。

**过渡过程：**

| 阶段 | 画面效果 |
|------|---------|
| 触发 | 当前场景边缘出现水彩颜色，像素抖动（dithering）扩散 |
| 覆盖 | 水彩逐渐覆盖整个画面，颜色像在湿纸上晕开，但扩散过程保持像素网格感 |
| 浮现 | 新场景从水彩颜色中"浮现"，像素从模糊到清晰 |

**不同目的地用水彩颜色区分：**

| 目的地 | 过渡颜色 | 传递的情绪 |
|--------|---------|-----------|
| 大屋出门 | 暖黄色 | 阳光、出门的兴奋 |
| 去鱼塘 | 青绿色 | 水、草、外公的世界 |
| 去集市 | 橙红色 | 热闹、赶集 |
| 回主角房间 | 蓝灰色 | 安静、私人空间 |
| 天台 | 深蓝色渐变到橙色 | 傍晚、开阔、呼吸感 |

**参考**：《Gris》的水彩过渡效果，但用像素化处理保持风格统一。

### 童年阶段场景

#### 1. 外婆大屋（主基地）

外婆大屋是游戏的第一个场景，也是主角在乡下的落脚点。三层红砖楼，旁边有一座废弃的黄土老屋。屋前有院子，出门就是村道。

这是整个游戏最重要的场景——主角在这里睡觉、吃饭、和家人互动、触发大量事件。它不仅是"一个地点"，是主角慢慢从"外来者"变成"家里人"的地方。

#### 大屋各层详情

**一楼 —— 生活核心区**

| 区域 | 描述 | 可互动元素 |
|------|------|-----------|
| **便利店** | 大屋临院的一侧开了个小窗口，摆着几个老旧的塑料罐——瓜子、花生、糖果、辣条、冰棍。外婆坐在窗口后面的小竹椅上，有人来买东西就站起来 | 用零花钱买零食、帮外婆看店、偷吃被抓事件 |
| **客厅/堂屋** | 一张大圆桌+几把椅子，墙上有挂历和全家福，角落有一台老式电视机。全家吃饭和聚集的地方 | 吃饭事件、看电视日常、家庭对话触发点 |
| **厨房** | 柴火灶，灶台是水泥砌的，灶膛里烧柴。旁边有水缸、碗柜、菜板。舅妈和外婆轮流做饭 | 帮忙烧火、看做饭的日常、学做菜事件 |
| **杂物间** | 堆着米缸、腌菜坛子、农具、换季的被子 | 帮忙拿东西的日常 |

**美术特征**：水泥地面，白石灰墙有些发黄，灶台的墙面被烟熏黑了。光线偏暗，但有烟火气。吃饭时间会有蒸汽和香味。

**二楼 —— 居住区**

| 区域 | 描述 | 可互动元素 |
|------|------|-----------|
| **外婆外公的房间** | 一张老式木床，蚊帐，床头柜上有外婆的针线盒。外公基本不住这里（守鱼塘），所以房间主要是外婆一个人的 | 去找外婆聊天、外婆生病躺在床上的事件 |
| **舅舅舅妈的房间** | 比较整洁，舅妈收拾的。有一台小风扇，墙上贴着大表哥以前的奖状（已经发黄了） | 舅舅喝了酒回来、舅妈骂他的日常 |
| **二表哥的房间** | 乱七八糟，床上堆着衣服，地上有零食袋。墙上贴着他画的画（画得不好但很多）。土狗经常趴在门口等他 | 去找二表哥一起玩、看到他房间的混乱场面 |

**三楼 —— 主角的楼层**

| 区域 | 描述 | 可互动元素 |
|------|------|-----------|
| **主角的小房间** | 以前是杂物间改出来的，不大，一张小木床+一张小桌子+一把椅子。窗外能看到村子的屋顶和远处的鱼塘方向。这是主角在乡下的"自己的空间" | 睡觉存档、想家事件、写作业、窗外发呆 |
| **天台** | 三楼楼顶是平的，可以上去。视野开阔，能看到整个村子、远处的稻田、鱼塘方向。傍晚的风很舒服 | 看日落、数星星、想心事的安静场景。少年阶段回乡时第一个去的地方 |

**美术特征**：三楼比一楼新一点（后来加盖的），但也不精致。主角房间的窗户是老式铁框窗，推开发出吱呀声。天台有简单的水泥护栏。

#### 院子

| 区域 | 描述 | 可互动元素 |
|------|------|-----------|
| **水井/水龙头** | 院子角落有口水井（老的），旁边接了一个水龙头（新的）。早上外婆在这里洗衣服、洗菜 | 帮外婆打水、洗菜、夏天冲凉水 |
| **鸡棚** | 几只土鸡在院子里散养，有个简易的鸡棚。鸡到处走、到处拉屎 | 追鸡搞笑事件、捡鸡蛋事件 |
| **晾衣绳** | 竹竿搭的晾衣绳，上面挂着衣服、被单。风一吹鼓起来像船帆 | 帮舅妈收衣服的日常 |
| **龙眼树** | 院子角落有一棵老龙眼树，夏天结满了龙眼。树下有阴凉 | 摘龙眼、树下乘凉、和小朋友在树下玩 |
| **杂物间（院子）** | 放农具、柴火、旧物 | 找东西的日常 |

**美术特征**：泥土地面，有些地方长了草。阳光充足的时候院子里很亮，鸡在走、衣服在飘、树在摇。下雨天泥泞、积水。

#### 黄土老屋（大屋旁的废弃老屋）

大屋紧挨着一座低矮的黄土瓦房，年代久远，以前是家里的厨房和柴堆房，后来盖了新楼就废弃了，现在堆杂物。

**外观**：黄土墙，瓦片屋顶有几处破损，门框很低矮（大人要低头进去）。比大屋矮很多，看起来像大屋的"影子"。

| 功能 | 说明 |
|------|------|
| **探索/发现** | 主角小时候喜欢钻进去探险，可能在里面找到旧物——褪色的老照片、外婆年轻时用的炊具、舅舅小时候的玩具 |
| **外婆的回忆** | 外婆偶尔一个人进去待一会儿，主角跟进去才发现她只是站在旧灶台前发呆。这里曾是她年轻时做饭的地方 |
| **阿宝的闯祸** | 阿宝在里面玩火、碰倒了东西、被小凤追着打到躲进去 |
| **躲雨** | 突然下雨时最近的避雨处就是这座老屋，可能触发和某人一起躲雨的事件 |
| **二表哥的秘密** | 二表哥在里面藏了东西——可能是漫画书、弹弓、或者不想让舅舅看到的东西 |
| **安静的空间** | 和大屋的热闹形成对比。主角想一个人待着的时候可以来这里。傍晚光线从破瓦缝照进来 |
| **时间的象征** | 旁边是三层新楼，旁边是低矮的黄土老屋。不用说一句话就讲完了这个家几十年的变化 |

# 07 - 模拟经营

## 经营系统的定位

经营元素是**角色属性成长的手段**，不是游戏的核心目标。玩家通过劳动获得属性加成，体验茂名农村生活。

---

### 家务类（外婆牵头）

|------|------|----------|--------|
| **择菜** | 帮外婆择菜/剥蒜 | 懂事+1, 亲密+1 | 简单、日常 |

### UI 设计原则

- **无常驻 HUD**：画面保持干净，不显示血条、地图、属性等任何常驻信息
- **属性不主动提醒**：属性变化不会弹出通知，玩家需打开菜单自行查看
- **漫画框式对话**：对话以漫画气泡的形式在角色旁边展现，非底部对话框
- **表情差分演出**：对话时展示角色的像素表情差分，参考《爱氏物语》

---

### 对话演出系统

#### 漫画框式对话

对话不是传统的底部对话框，而是像漫画一样在角色旁边出现气泡框，配合角色表情差分。

**平时画面（无对话）：**
```
┌────────────────────────────────────────┐
│                                        │
│           （完全干净的场景画面）          │
│                                        │
│                                        │
│          外婆            主角           │
│                                        │
└────────────────────────────────────────┘
```
- 画面无任何 UI 元素
- 靠近可互动物体时出现小提示（可选）

**对话画面：**
```
┌────────────────────────────────────────┐
│                                        │
│    ┌──────────────────────┐            │
│    │ ┌────┐               │            │
│    │ │ :) │ 乖孙，过来帮   │            │
│    │ └────┘ 奶奶择菜。     │            │
│    └──────────────────────┘            │
│         外婆            主角            │
│                                        │
│    ┌──────────┐  ┌──────────────┐      │
│    │好的奶奶！│  │我想出去玩…… │      │
│    └──────────┘  └──────────────┘      │
└────────────────────────────────────────┘
```
- 对话气泡从说话角色旁边冒出
- 气泡内包含角色表情差分 + 对话文本
- 选项也是漫画风格的小气泡，非传统按钮

#### 对话框类型

| 类型 | 样式 | 适用场景 |
|------|------|---------|
| **普通气泡** | 小框，角色头顶或身旁 | 日常对话、短句 |
| **放大框** | 框更大，文字更多 | 长段对话、讲故事 |
| **特写框** | 镜头拉近，半身表情占画面 | 重要剧情、情感高潮 |
| **内心框** | 异形框（云朵形/锯齿形） | 主角内心想法 |

#### 选项呈现

- 选项不是底部按钮列表，是漫画风格的选择气泡
- 每个气泡一句话，从主角旁边冒出
- 用方向键或点击选择
- 选中时气泡高亮放大

---

### 角色表情差分

每个角色需要一套像素表情差分，在对话时展示，参考《爱氏物语》的风格。

#### 基础表情列表

| 表情 | 触发场景举例 |
|------|------------|
| **普通** | 日常对话、没有特别情绪 |
| **微笑** | 开心、欣慰、温馨时刻 |
| **大笑** | 搞笑场景、被逗乐 |
| **皱眉** | 不满、担心、思考 |
| **生气** | 骂人、发飙（舅舅、小凤） |
| **难过** | 离别、委屈、心疼 |
| **惊讶** | 意外事件、震惊 |
| **害羞** | 被夸、不好意思（小英常见） |
| **无语** | 阿宝又闯祸了、无奈 |

| 角色 | 需要的特殊表情 |
|------|--------------|
| 外婆 | 普通、微笑、担心、心疼、唠叨 |

- 风格保持像素感但细节更丰富（眼睛高光、面部微表情）

# 人物档案

## 外婆

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 外婆 |
| **年龄** | 60+ |
| **身份** | 外祖母，家里的精神支柱 |

### 人物小传

外婆是这个家的根。她一辈子没离开过这片土地，外公、舅舅、两个孙子，再加上一个从县城送来的小外孙，全部靠她一个人的耐心撑着。

她老了，重活已经干不动了，但一日三餐从来没落下过。灶台是她的战场，厨房是她说了算的地方。她做饭不精致，但量大、管饱、有家的味道。

大屋一楼有一角是她开的小便利店，用一些老旧的塑料罐子装着零食来卖——瓜子、花生、糖果、辣条，还有小孩爱吃的零碎东西。赚不了几个钱，但她觉得闲着也是闲着，有人来买还能顺便聊几句。罐子是攒了很多年的，有的发黄了，有的盖子裂了用橡皮筋箍着。主角有时候会蹲在罐子前面挑零食，外婆就在旁边看着，偶尔说"这个辣，你吃不了"。

她对主角是真心疼的。这个孩子不是自己生的，但来了就是自己家的。她会偷偷给主角塞零食，会在其他大人不在的时候多说几句贴心话。但她的精力有限，要操心的事情太多，有时候顾不上主角的情绪——不是不关心，是真的忙不过来。

### 性格特点

- **温和絮叨**：嘴上总在说"吃饱了没有""多穿点""别跑太快"，是那种用唠叨表达爱的人
- **传统**：信老规矩，逢年过节该拜的拜、该做的做，年例的时候她比谁都认真
- **有耐心**：对几个孩子都很包容，不打不骂，最多叹口气
- **隐忍**：家里的难处她都知道——舅舅的心灰、舅妈的辛苦、外公的沉默——但她不说破，默默撑着

### 说话方式

- 语速慢，带着茂名白话的腔调
- 口头禅："乖孙""食咗饭未啊"（吃了没有）
- 对主角的称呼随心情变：叫名字、叫"乖孙"、叫"细佬仔"（小孩子）
- 偶尔冒出一两句老话、谚语

### 与女儿（主角妈妈）的感情

女儿嫁出去了，去了外地，老公也得出去打工，日子过得不容易，孩子没人带，只能送回来。

外婆心疼女儿，但心里也有一个说不出口的想法：你小时候我带你，老了还要帮你带你的孩子。但她说不出来。她不会跟女儿抱怨，也不会跟主角说"你妈不要你了"这种话。她就是接着，默默接着。

逢年过节女儿打电话回来，她会说"孩子好着呢，你放心"，然后挂了电话叹口气。

她对主角的好，有一部分是对女儿的牵挂——你照顾好她的孩子，就是帮了她最大的忙。主角是外婆和女儿之间的情感纽带，外婆照顾主角，某种程度上也是在替女儿尽母亲的责任。

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 主角刚到，对外婆有距离感，想家 |
| **发展** | 日常相处积累亲密感，择菜、做饭、赶集 |
| **关键转折** | 某个主角特别想家/受委屈的夜晚，外婆的安慰 |

> 外婆是主角在乡下最先建立情感联系的人。她不会说什么大道理，但她的存在本身就是一种安全感。

### 参与事件

- 择菜、做饭、赶集（日常活动）
- 年例准备（拜神、备菜）
- 台风天（关窗、收东西）
- 主角想家时的安慰事件

### 视觉设计参考

- 穿着：深色碎花布衫、黑布裤、塑料拖鞋
- 体态：微驼背、走不快、手上有老茧
- 表情：大部分时候是慈祥的笑，偶尔叹气时眉头微皱

## 外公

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 外公 |
| **年龄** | 60+ |
| **身份** | 外祖父，鱼塘和猪圈的守塘人 |

### 人物小传

外公是这个家最沉默的人。

他在鱼塘边有一座自己的小瓦砖房，旁边是猪圈，房前种了几棵芒果树。他不像其他人住在外婆家的大屋里，而是长年一个人守在这片产业上。喂鱼、养猪、巡塘、修补——日复一日。

他是这个家真正的老黄牛。鱼塘和猪是家里主要收入来源之一，外公一个人撑着这块。他不抱怨，也不多说话，但如果你仔细看，他的手比谁的都粗糙，脸比谁的都黑。

他的沉默不是天生的。年轻的时候，他被卷入过一场南边的仗。他不是正式的军人，但那场战争还是在他身上留下了痕迹。从那以后，他就变成了一个不爱说话的人。他不再跟人解释什么，也不再对事情发表看法——做就是了，别问为什么。

他对主角不会主动亲近。他不哄孩子，也不逗孩子。但如果你去鱼塘找他，他会默默教你怎么做——怎么撒饲料、怎么看鱼浮上来的样子、怎么判断天会不会下雨。他不说"我教你"，他直接做，你看不看、学不学，是你的事。

他住在鱼塘边不是因为跟外婆有什么矛盾，而是他觉得住在这里最方便看塘。鱼塘是家里的饭碗，他守着饭碗，天经地义。鱼塘离大屋很近，他早上有时候会回大屋一趟——手里提着一桶小鱼，放到主角面前，让主角自己挑几条去养。他不说这是给你的，他就是把桶放下。芒果熟了也会摘几个带回来，往桌上一放就走。

这就是他的爱的方式——不解释，不邀功，只是默默把东西带到你面前。

### 性格特点

- **沉默寡言**：一天说的话可能不超过十句，能用动作表达就不开口
- **实干**：不讲虚的，鱼塘的事、猪圈的事，他亲力亲为
- **有威严**：不发火，但家里人都怕他。舅舅在外面算个生意人，回来在老爹面前一样老实
- **独立**：不住大屋，自己守着鱼塘瓦房，不给任何人添麻烦
- **手艺好**：修补鱼塘、搭猪圈、砍芒果树枝，什么都会做
- **被战争改变过**：年轻时卷入过一场南边的仗（非正式军人），回来后性格大变，从那以后不再多说话

### 年轻时的外公

外公年轻的时候跟现在判若两人。他被卷入过一场南边的仗，不是作为正式的军人，但那场战争在他身上留下了不可逆的痕迹。具体经历了什么，他从来没跟任何人说过——包括外婆。

从那以后，他不再跟人解释什么，不再对事情发表看法。他把所有精力投到了鱼塘和猪圈上——也许是因为只有这些实实在在的活，才能让他不去想那些事。

> **游戏内模糊化表达**：不直接提及"对越自卫反击战"，用"南边的仗""那年的事""从战场回来以后"等模糊表述。玩家不需要知道具体是什么战争，只需要感受到一个被改变过的老人。

### 说话方式

- 话极少，经常只说半句："去那边""看着""嗯"
- 不说教，不讲道理，用行动示范
- 对主角偶尔蹦出一句让人记住很久的话，比如"鱼要慢慢喂，急不得"

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 主角觉得外公很可怕，不敢靠近 |
| **发展** | 主角主动去鱼塘找外公，外公默默接受这个"小跟班" |

### 与舅舅的关系

## 舅舅

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 舅舅 |
| **年龄** | 35-40 |
| **身份** | 舅舅，曾在城里做生意，现在帮外公干农活 |

### 人物小传

舅舅是这个家最复杂的人。

他曾经在茂名市区有自己的生意，是一个见过世面的人。但事业上遭遇了不如意——具体什么原因可以后续展开——他带着舅妈回到了乡下。

回到乡下后，他帮外公打理鱼塘和果林。活是干的，但心是灰的。他不像外公那样心无旁骛地守着产业，他干着活的时候可能在想别的事——那些没做成的生意、没赚到的钱、没实现的野心。

他在家里是"不上不下"的位置。对外公他是儿子，要听话；对舅妈他有愧疚——人家跟着自己从城里来到这乡下地方；对两个儿子他可能不知道怎么面对——自己都活不明白，拿什么教育孩子。

但他不是坏人。他严厉，是因为他对自己也严厉。他不善于表达关心，是因为没人教过他怎么表达。如果家里遇到大事——比如台风、比如需要做重要决定——他是会站出来的那个人。

### 性格特点

- **严厉**：对孩子们要求高，但不会好好说，总是用批评的语气
- **疲惫**：不是身体累，是心累。从城里退回来的中年人都有这种疲惫
- **不善表达**：关心人的方式是默默干活、把事情做好，而不是说好听的话
- **有担当**：平时看着消沉，真遇到事他是能扛的
- **敏感**：别人一句无心的话可能戳到他的痛处，尤其是关于"城里"的话题

### 说话方式

- 语气偏硬，不是吵架的硬，是不知道怎么柔软下来的那种硬
- 不怎么说过去的事，关于城里的生意只字不提
- 对主角说话简短直接："去帮忙""别乱跑""吃饭了"
- 喝了酒之后可能会多说几句，但说的也都是牢骚

### 与主角的关系

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 舅舅对主角客气但疏远，多一张嘴吃饭而已 |
| **发展** | 主角跟在舅舅后面干活，舅舅偶尔教一两句，态度慢慢软化 |
| **关键转折** | 某次舅舅喝了酒或者遇到什么事，对主角说了一些真心话 |

> 舅舅是主角理解"大人也不容易"的窗口。他不是英雄，也不是反派，只是一个被生活打败了但还在硬撑的人。

### 与舅妈的关系

舅舅和舅妈结婚很多年了，日子过得不算富裕，但也不是没有温度。

两人相处的日常模式是斗嘴。舅妈嘴上从来不饶人，嫌他好吃懒做、嫌他干农活磨蹭、嫌他一天到晚想那些没用的。舅舅被骂了不还嘴，一笑了之，有时候还故意逗她。舅妈越骂越来劲，舅舅就越笑越开心——斗嘴是他们之间最常见的情感表达方式。

舅舅对舅妈有愧疚。人家跟着自己从城里来到这乡下地方，没有享过什么福，净吃苦了。但他不会说"对不起"这种话，他表达愧疚的方式是：舅妈骂他的时候他不还嘴，舅妈累了的时候他多干一点活，舅妈发火的时候他笑一笑把火气接住。

在村里，舅舅是另一副面孔。他的人缘很好，大屋门口看到有人经过就热情地打招呼，"来坐啊""进来喝杯茶"。他喜欢热闹，喜欢家里有人来。舅妈一边骂他"又叫人来，饭都不够吃"，一边去厨房多炒两个菜。

### 与外公的关系

舅舅和外公是这个家里最沉默的一对父子。

外公是"做不说"的人，一辈子不解释、不评价。舅舅是"说但没做成"的人——在城里做过生意，见过世面，性格比外公话多、主意多、脾气也更外放。但回到乡下后，他在老爹面前收敛了。

外公从头到尾不评价舅舅城里的失败，一个字不提。但这种沉默比任何责骂都让舅舅难受——沉默可以被解读为"我早就知道你会这样"。舅舅偶尔会冒出一些"城里怎样怎样"的话，或者对鱼塘的做法有自己的想法，但外公不接茬，继续按自己的方式干。舅舅觉得老爹听不进去，外公觉得你连自己都顾不好还想教我。

两人不吵架，但暗涌不断。真遇到大事——台风来了、鱼塘出问题——舅舅是能扛的，外公也知道。父子俩平时各干各的，危机时刻不用说就站到一起了。

### 与两个儿子的关系

舅舅对两个儿子的态度，是他心里最拧巴的部分。

大表哥最像年轻时的舅舅——不爱学习，敢闯荡，有野心。舅舅一眼就认出了自己年轻时的影子，但这让他更害怕。他知道那条路有多难走，他自己走过，走不通。所以大表哥每次从县城回来，舅舅总要刺他几句："在饭馆能学出什么名堂""你以为开饭店那么容易"。大表哥不服，觉得舅舅是被打败了才回来的，凭什么说我。父子俩经常吵架，一吵就崩，谁都不让谁。

二表哥则像现在的舅舅——不上进，安于现状，待在乡下不觉得有什么不好。舅舅对二表哥反而严厉，因为他害怕二表哥变成现在的自己——沉沦在乡下，一辈子就这样了。但二表哥不像大表哥那样顶嘴，舅舅骂他他就听着，听完该干嘛干嘛，舅舅的火发不出来，闷在心里。

> 大表哥是"以前的舅舅"，二表哥是"现在的舅舅"。舅舅看着大表哥像是看到了要走自己老路的儿子，看着二表哥像是看到了已经走完这条路的自己。两个都让他心疼，但他一个都说不出口。

## 舅妈

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 舅妈 |
| **年龄** | 35-40 |
| **身份** | 舅舅的妻子，从城里跟来乡下，家里的"发动机" |

### 人物小传

舅妈是这个家最踏实的人。

她跟着舅舅从城里来到乡下，没有一句怨言。不是忍着不说，是真的不觉得有什么好抱怨的。在她看来，日子就是过的，在哪过不是过，一家人在一起比什么都强。

她勤劳、踏实、肯干、能吃苦。家里家外一把抓——操持家务、养猪养鱼养果树，没有她不干的活。外婆年纪大了，她主动接过了大部分家务和农活；外公守鱼塘，她帮忙喂猪打扫猪圈；舅舅心灰意冷，她也不催他、不怨他，该干的活自己先干了。

她对主角的外公外婆照顾得细心体贴，当自己亲生父母一样待。外公外婆也把她当自己女儿看——这个家能转起来，一半靠外婆的耐心，一半靠舅妈的能干。

她对两个儿子的态度是：爱但不强求。大表哥和二表哥都不爱读书，初中就辍了学。舅妈没有责骂，也没有哭天抢地。她想的是：孩子能快乐地生活就好，不是每个人都得走读书那条路。大表哥去县城学厨艺，她支持；二表哥留在乡下，她也不嫌弃。在她眼里，两个孩子都是好的。

她和主角的妈妈关系很好。主角的妈妈常说："你舅舅能娶到这样的女人，是他一辈子的福气。"舅妈对主角的照顾，有一部分是因为和主角妈妈的情分——你是她孩子，我待你跟待自己孩子一样。

### 性格特点

- **勤劳**：从早到晚不停，家务农活两头跑，是这个家最能干的人
- **善良**：对老人孝顺，对孩子慈爱，对主角视如己出
- **踏实**：不抱怨、不攀比、不计较，日子该怎么过就怎么过
- **能吃苦**：从城里来到乡下，干的活比谁都多，从不叫苦
- **豁达**：对两个儿子辍学不强求，觉得快乐生活就好

### 与舅舅的关系

舅妈嘴上从来不饶人，嫌舅舅好吃懒做、嫌他干农活磨蹭、嫌他一天到晚想那些没用的。但骂归骂，她心里从来没有真的嫌弃过舅舅。跟着他从城里到乡下，吃苦受累，她觉得值得。

舅舅被骂了一笑了之，有时候还故意逗她。舅妈越骂越来劲，舅舅就越笑越开心。斗嘴是他们之间最常见的情感表达方式，比说"我爱你"更像他们。

外人看着觉得舅妈吃亏——跟着一个失败的男人来到乡下。但舅妈自己不这么想。她觉得舅舅不是坏人，只是运气不好。他有本事的时候她跟着享过福，他落魄了她也跟着吃得起苦。这就是夫妻。

### 与主角的关系

舅妈对主角的照顾是真心实意的。你来了就是这个家的孩子，跟大表哥二表哥一样，该吃饭吃饭，该穿衣穿衣，不会亏待你。

主角的妈妈和舅妈关系好，舅妈对主角也带着这份情分。但不全是情分——舅妈本身就是一个善良的人，对谁都不会差，何况是自家的孩子。

> 舅妈是主角在这个家里"不需要争取就获得的善意"。和外婆的温情不同，舅妈的好是实在的、不带太多感情色彩的——她就是对你好，因为她就是这样的人。

### 说话方式

- 语速快，干活的时候边干边说
- 不说废话，吩咐事情干脆利落："去把鸡喂了""菜端上去"
- 对舅舅："你又在那想什么，过来帮忙"
- 对主角偶尔关心一句："吃饱了没？""别跑太快，摔了"

### 参与事件

- 家务活动（择菜、扫地、喂鸡）
- 养猪养鱼养果树（农活活动）
- 赶集（和外婆一起去）
- 年例备菜
- 舅妈和舅舅斗嘴（日常事件）

### 视觉设计参考

- 穿着：朴素但利落，干活时袖子永远卷到手肘
- 体态：结实、不胖，操劳的人没有胖的
- 表情：干活时认真，闲下来会笑，眼角有皱纹

## 大表哥

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 大表哥 |
| **年龄** | 18 |
| **身份** | 舅舅的大儿子，在县城学厨艺，周末/节假日回乡下 |

### 人物小传

大表哥是这个家最"往外走"的人。

他初中辍了学，先去外地的电子厂打过工，后来回到茂名，在县城里跟一位师傅学粤菜烧腊。师傅对这个徒弟很满意——踏实肯干，不偷懒，不抱怨。大表哥心里憋着一个没跟家里人说过的野心：将来要在茂名开一间自己的大排档或小饭馆。

他为什么不说？因为他见过舅舅是怎么被"城里"打败的。如果他说"我要去城里闯"，家里人（尤其是舅舅）会怎么想？所以他只做不说，在县城默默学手艺，周末回来给家里做一顿饭。他的手艺在慢慢进步——第一次做的菜可能不怎么样，但每一次回来都比上一次好一点。

他继承了舅妈的优点——踏实、肯干、能吃苦。也继承了舅舅的社交能力——开朗、重情意、会来事。在县城里跟师傅、跟同行、跟饭馆的客人，他都能处得来。

他是主角认识"外面世界"的窗口。每次回来，他会给主角带点县城的东西——一包零食、一本漫画、一个小玩意。这些东西在乡下买不到，对主角来说就是"外面的证据"。他也会跟主角讲县城的见闻，但不是炫耀，而是分享——他把主角当一个可以聊天的人，当亲弟弟一样关爱。

### 性格特点

- **踏实肯干**：继承了舅妈的品质，干活不偷懒，师傅也满意
- **敢闯荡**：去过外地电子厂，现在学手艺，不满足于待在乡下
- **开朗重情意**：继承了舅舅的社交能力，为人仗义
- **有野心但低调**：想开饭馆但不张扬，只在跟主角聊天时偶尔提一句
- **倔**：跟舅舅一样倔，不肯服软

### 与舅舅的关系

大表哥和舅舅是这个家里最拧巴的一对父子。

舅舅从城里退回来，大表哥往城里闯。大表哥以前觉得舅舅是被打败了才回来的，凭什么说我。舅舅觉得大表哥迟早也会撞南墙。每次回来，舅舅总要刺他几句，大表哥不服，父子俩经常吵架，一吵就崩，谁都不让谁。

但在外闯荡过后，大表哥渐渐理解了父亲的不易。他开始明白那条路确实不好走，舅舅不是懦弱，是真的拼过。但他不会跟舅舅说这些话，他和他爸一样倔，不肯服软。这份理解藏在心里，也许要很多年后才会说出来。

### 与舅妈的关系

大表哥对舅妈这位母亲十分敬重。舅妈是这个家最不抱怨的人，跟着父亲从城里来到乡下，吃苦受累，从来不叫苦。大表哥看在眼里，心里是服气的。他对舅妈的好是实打实的——每次回来给家里做顿饭，首先就是做舅妈爱吃的菜。

舅妈对他辍学的事没有责骂，只说"你开心就好"。大表哥表面上不说什么，心里是记着的。

### 与二表哥的关系

大表哥对弟弟的感情很复杂。

一方面，他和对父亲一样怒其不争——二表哥不上进，整天在乡下混日子，没有一点想出去闯的意思。大表哥偶尔会帮他找一些工作机会，希望他能更上进一些。

另一方面，他又觉得二表哥活得比自己轻松。二表哥没有野心，没有压力，日子过得简单快乐。大表哥有时候会想：也许二表哥才是对的？

他不会跟二表哥说这些。他只会说："有空也出来找点事做。"二表哥说"不急"，他就不再多说了。

### 与主角的关系

大表哥对主角像对亲弟弟一样。他是主角认识外面世界的窗口，每次回来带东西、讲故事、教东西。他把主角当成一个可以倾诉的人——想开饭馆的野心，第一次就是对主角说的。

> 大表哥是主角的"榜样型玩伴"——他比主角大，见过更多东西，有自己想做的事。他让主角看到"不读书也有出路"，但也让主角思考"那条路是不是我想要的"。

### 与外公外婆的关系

大表哥对外公外婆很有孝心。每次回来给家里做顿饭，也给老人做他们喜欢的菜。他知道外婆年纪大了操持家务不容易，舅妈扛起了大部分，但他能做的也会做。对外公，他话不多，但偶尔会去鱼塘看看，带包烟或者带点吃的。

### 关键弧线

大表哥的做菜手艺是可视化的成长线：
1. 第一次回来做的菜，味道一般，但全家人都给了面子
2. 中间几次，有进步但也有翻车（比如盐放多了）
3. 某一次做了一道真正好吃的菜，全家人都惊讶
4. 最后阶段，他做的菜已经像模像样，甚至外婆都点头认可

这个过程不需要玩家做什么，只需要"在场"——看着他变化。

### 说话方式

- 在主角面前话多，愿意分享、愿意教
- 在大人面前话少，尤其是舅舅面前
- 口头禅："没啥大不了的""等我以后……"
- 偶尔带点县城的词或者饭馆里的行话（"火候""出菜"）

### 参与事件

## 二表哥

### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 二表哥 |
| **年龄** | 16 |
| **身份** | 舅舅的小儿子，留在乡下，主角最亲近的玩伴 |

### 人物小传

二表哥是这个家最快乐的人，也是最像舅舅的人——长得像，性格也像现在的舅舅：开朗、爱笑、爱玩、不想太远的事。

他跟大哥一样初中辍了学。但跟大哥不同的是，他完全没有"我要学手艺""我要去城里"这种想法。他的世界就是这个村子——鱼塘边的泥巴、果林里的鸟窝、村口的竹林、河沟里的蝌蚪。他在家里养了一只土狗和一只狸花猫，每天带着狗在村里到处跑。

他是主角来到乡下后第一个真正玩到一起的人。大表哥不常在家，大人们各有各的事，只有二表哥是随时可以叫出来疯的。他很乐意带这个比他小得多的弟弟去钓鱼、去玩游戏、去各种主角不敢去的地方——钻进果林深处、跑到鱼塘边抓青蛙、翻过田埂去看邻村的孩子放牛。

他皮，但不是熊孩子那种皮。他不做坏事，只是精力旺盛、坐不住。他对主角是真心仗义的——谁欺负主角他第一个冲上去，有好吃的会分一半给主角。

他平时也会帮家里干活，喂鸡、扫地、跑腿，干得不比别人少。村里很多人都认识他，因为他一天到晚在村里晃，见谁都笑嘻嘻打招呼。

他不喜欢去城里，甚至有点抗拒。大表哥有时候说"来县城看看"，他总是找理由不去。也许是自卑——觉得自己什么都不行，去城里不知道能干什么。但在乡下他就是王，哪里有好玩的、哪里有好钓鱼的，他门儿清。

> 二表哥是"现在的舅舅"——安于现状，不争不抢，在乡下过自己的日子。但他比舅舅快乐，因为他还没有经历过失去。

### 性格特点

- **开朗爱笑**：长得像舅舅，性格也像，整天乐呵呵的
- **皮**：坐不住、跑不停、永远有使不完的劲
- **有爱心**：养了一只土狗一只狸花猫，对动物很好
- **仗义**：对朋友好，有东西分一半，有人欺负你他帮你出头
- **天不怕地不怕**：不害怕虫子、不害怕黑、不害怕外公的黑脸
- **大大咧咧**：被骂了转头就忘，不记仇、不纠结
- **有点自卑**：不喜欢去城里，觉得自己在那里什么都不是

### 与舅舅的关系

二表哥一直住在大屋里，跟舅舅抬头不见抬头见。舅舅对他有点严厉——不是大表哥那种吵架式的对抗，而是更拧巴的一种严厉。他其实有点害怕二表哥和自己一样，沉沦在乡下，一辈子就这样了。他在二表哥身上看到了自己的影子，这让他焦虑。但二表哥不像大表哥那样顶嘴，舅舅骂他他就听着，听完该干嘛干嘛，舅舅的火发不出来，闷在心里。

### 与舅妈的关系

舅妈对二表哥格外关爱。大表哥有本事，敢闯荡，有手艺，舅妈不担心他。但二表哥看上去没什么特长，在乡下晃来晃去，舅妈看在眼里，疼在心里。她不会说什么重话，只是默默多照顾一些——好吃的多给他留一份，天冷了提醒他加衣服。二表哥知道妈妈对他好，偶尔也会主动帮舅妈干点活。

### 与大表哥的关系

大表哥偶尔会帮二表哥找工作机会，希望他能更上进。二表哥每次都说"不急"，然后就岔开话题。他知道大哥是为他好，但他真的不知道出去能干什么。在乡下他什么都行，去城里他什么都不行。

他其实挺佩服大哥的——敢出去闯，有手艺，有想法。但他不会说出来。

#### 小明

#### 基本信息

| 项目 | 内容 |

#### 人物小传

小明是主角到村里的第一天就认识的人。

他家在村东头，离外婆家不远，门口种着几棵木瓜树。主角刚到的那天下午，他就跑过来看热闹——"这是谁家的孩子？"——然后不到十分钟就跟主角混熟了。

他是那种典型的粤西农村小男孩：皮肤晒得黑黑的，精力旺盛，坐不住，一天到晚在村里晃。他什么都敢试——爬树、摸鱼、捉知了、用竹子做弹弓。他不一定样样精通，但他什么都敢上手，失败了也不怕丢脸。

他是主角在村里的"引路人"。二表哥虽然也带主角玩，但二表哥是自家人，跟主角之间有辈分关系。小明是第一个让主角觉得"我在这个村子里有朋友了"的人。他主动拉主角加入村里的小朋友群体，介绍阿旺阿宝兄弟、小英小凤给主角认识。

他在小朋友中间有点小带头的意思，不是因为他最厉害，而是因为他最稳当。谁打架了他去劝，谁被欺负了他去帮忙，谁有好玩的他第一个喊人。他有一种天然的号召力。

他看起来和其他孩子没什么不同，但他的底色比别人硬。爸爸在他很小的时候就不在了——农村里常见的那种意外，游戏里不具体交代。妈妈一个人带他，种地、干家务、养孩子，不抱怨，也不溺爱。小明从小就知道自己是家里唯一的男孩子，他要替妈妈撑着。

所以他的动手能力强不是因为天赋，是因为没人帮他——家里什么东西坏了，妈妈不会修，他来。挑不动的水他学着挑，拧不开的螺丝他学着拧。他比别的孩子早熟，不是因为懂事的教育，是因为生活没有给他撒娇的余地。

但他在朋友面前从来不提这些。他永远是那个笑嘻嘻的、带头疯玩的小明。他的坚强不是装出来的，是他真的学会了——哭完了还是得回家帮妈妈做饭。

#### 性格特点

- **坚强**：不是天性，是被生活教会的。哭没有用，解决问题才有用
- **动手能力强**：家里没人帮忙，什么都得自己来。修东西、做工具，样样能上手
- **讲义气**：妈妈教他的——"对人要真心"。朋友有难第一个冲上去
- **爱出头**：习惯了当"家里的顶梁柱"，出去也忍不住站出来
- **嘴硬心软**：嘴上说"我才不怕"，其实心里也发怵，但为了面子死撑

#### 说话方式

- 语气干脆利落，像个小大人
- 喜欢说"走！""跟我来！""这个我会！"
- 被夸了会装作不在意："这有什么难的"
- 偶尔会吹牛："我上次抓到一条这么大……"（手比划的越比越大）
- 提到家里的事会轻描淡写带过，不让人觉得他特殊

#### 与二表哥的关系

小明和二表哥很熟，整个村就这么大，从小一起玩到大的。两人关系不错但也有较劲的时候——都是那种爱玩爱闹的性格，谁也不服谁。二表哥觉得小明太爱出头，小明觉得二表哥仗着年纪大不把人放眼里。偶尔会拌嘴，但很快又和好。

在带主角玩这件事上，两人有点"竞争"的意思：二表哥觉得主角是自己表弟，自己带就行；小明觉得多一个人多一份热闹，干嘛不让主角跟大家一起玩。

#### 与阿旺阿宝的关系

小明和胖兄弟是从小玩到大的邻居。他对阿宝经常翻白眼——阿宝总是惹祸，擦屁股的活经常是小明干。但他打心底把两兄弟当铁哥们。

#### 与母亲的关系

小明的妈妈是这个村子里最沉默的寡妇之一。

她不是那种悲悲戚戚的人——不哭诉、不抱怨、不跟人诉苦。她每天天不亮就起来干农活，晚上回来做饭，日复一日。她对小明的方式是：你是个男孩子，你要学会自己照顾自己。不打不骂，但也谈不上温柔。

小明知道妈妈辛苦。他从不跟妈妈要东西、从不闯祸让妈妈操心。他帮妈妈干活的时候不说"我来帮你"，而是直接上手——劈柴、挑水、扫地。母子之间的爱不需要语言，是用行动堆出来的。

但小明也有脆弱的时候。比如看到别的孩子被爸爸扛在肩膀上、比如过年别人家放鞭炮他家安安静静。他不会哭出来，只是愣一下，然后转身继续干活。

> 妈妈没有给小明一个完整的家，但给了他一副硬骨头。这副骨头让他以后能去参军、能扛枪、能保家卫国。

#### 阿旺

#### 基本信息

| 项目 | 内容 |

| **昵称** | 阿旺 |
| **年龄** | 8 |

#### 人物小传

阿旺是村里出了名的"大胖"。不是那种虚胖，是结实的胖——吃得下睡得着，力气也大。他比弟弟阿宝大一岁，但看起来像大了两三岁。

他家在村里算条件好的。爸爸在外面搞工程承包，赚得比种地的多不少。家里是村里少有的贴了瓷砖的二层小楼，阿旺阿宝的零花钱也比别的孩子多。但这些在小孩子的世界里没有那么明显——阿旺不炫耀，他只是口袋里的零食从来不断、衣服比别人新一点。

他话不多。在一群小朋友吵吵闹闹的时候，他经常站在旁边安安静静地吃东西。他不爱抢着说话，但说出来的话往往比别人靠谱。

他是那种"平时不吭声，关键时刻靠得住"的人。一起玩的时候他可能是最不起眼的那个，但遇到事——比如有人掉进水沟、比如迷路了、比如天黑了害怕——他是最冷静的那个。他会说"别慌"，然后想办法。

他的安静有一部分是习惯了。爸爸常年在外面跑工程，妈妈在家也不怎么管他——不是不爱，是觉得"这孩子不用操心"。家里不缺吃不缺穿，但也没人陪他说话。他从小就学会了自己跟自己待着，一个人安安静静地吃东西、发呆、看天。所以他对食物有一种特殊的情感——吃东西的时候他是满足的、安全的。

他对弟弟阿宝有一种本能的保护欲。阿宝冒冒失失闯了祸，他不骂，只是默默跟在后面收拾。别人欺负阿宝，他二话不说站上去。他不跟人吵架，但他那个体型往那儿一站，很多事就解决了。

#### 性格特点

- **沉默寡言**：不爱说话，但不是那种闷，是懒得说
- **憨厚老实**：不耍心眼，说什么就是什么
- **贪吃**：对食物有执念，闻到香味就会出现在饭桌旁
- **可靠**：关键时刻比任何人都稳得住
- **护弟**：对阿宝有一种本能的保护，不许别人欺负他

#### 说话方式

- 话少，经常用"嗯""哦""行"就打发了
- 偶尔蹦出一句让全场安静的话，因为说到了点子上
- 说到吃的会突然话多起来："这个好吃""还有吗"
- 被问意见时经常说"都行""随便"，但心里其实有想法

#### 与阿宝的关系

阿旺对弟弟的态度是：外面的事我扛着，你别怕。

阿宝闯祸他不骂，只是默默善后。阿宝被人笑话胖，他挡在前面。阿宝话多到烦人，他也不嫌。两兄弟之间有一种不用说出来的默契——阿宝在外面再皮，到了哥哥面前就安静下来。

但阿旺也有一点小委屈：所有人都觉得他是"阿宝的哥哥"，没人注意他自己的感受。他不说，不代表不在意。

#### 与家人的关系

阿旺的爸爸在外面搞工程承包，一年大部分时间不在家。他不是那种不管家的爸爸——他会寄钱回来，过年回来会带新衣服和玩具——但他不了解两个孩子每天在干什么、在想什么。

妈妈在家，日子过得比村里其他女人轻松——不用下地干农活，偶尔打打牌、串串门。她对两个孩子的方式是"放养"：不缺吃不缺穿，但也不太管。阿旺考了多少分她不太在意，阿宝闯了什么祸她也只是说两句。

这种"物质充裕、关注不足"的环境，造就了阿旺现在的样子：不争不抢、不吵不闹，自己跟自己玩。他从不羡慕别人有什么——因为该有的他都有。但他有时候会羡慕别人家吃饭时吵吵闹闹的样子，自己家安安静静的，三个人各吃各的。

> 阿旺家和舅舅家是另一种对照：舅舅家没钱但有温度，阿旺家有钱但冷清。游戏不评断哪种更好，只呈现。

#### 人物小传

阿宝是这个小团体里的"事故制造机"。

他跟哥哥阿旺一样胖，但性格截然不同。阿旺是安静的胖，阿宝是闹腾的胖——他跑起来浑身的肉都在抖，摔倒了比谁都响，爬起来比谁都快。他一天到晚不是在惹祸就是在惹祸的路上。

他的口头禅是"我跟你们说"，然后开始讲一件根本没人想听的事。他话特别多，多到阿旺会默默走开、小明会翻白眼、小凤会直接吼他"你闭嘴"。但他完全不在意，下次继续说。

他最经典的模式是：说大话→尝试做→搞砸了→哭两声→马上忘了→继续玩。比如他说"我能游过那条河"，然后扑通跳下去差点溺水，被捞上来哭了五分钟，然后说"我明天再来"。

但他不是熊孩子。他的闯祸都是笨手笨脚造成的，不是故意捣乱。他心地很好——看到蚂蚁搬家会蹲着看半天，捡到钱会交给大人，别人哭了他会第一个凑上去问"你没事吧"。

他的无忧无虑是有底气的。家里条件好，他从小就不知道"缺"是什么感觉——零食永远有、衣服永远是新的、过年压岁钱比别人多。他不用像别的孩子那样帮家里干农活，也不用考虑"以后怎么办"。所以他可以一天到晚傻乐。但这种"无忧"也有另一面：没人告诉他什么该做什么不该做，没人帮他收着点，他的莽撞和冒失有一部分是因为从来没被认真管过。

他是整个小团体的快乐源泉。大家都习惯了他出丑，也都习惯了在他出丑之后笑着把他拉起来。

#### 性格特点

- **话多**：永远在说话，不说话会难受
- **冒失**：笨手笨脚，经常摔跤、撞到东西、弄坏东西
- **乐天**：被骂了转头就忘，被笑了也不生气
- **胆小**：嘴上天不怕地不怕，真遇到事第一个缩
- **善良**：心软，看到别人难过会手足无措地安慰

#### 说话方式

- 语速快，像倒豆子一样

- 闯祸之后的第一句话永远是："我不是故意的……"
- 被骂了之后："哦知道了下次不会了"（下次还会）

阿宝对家的感觉是：什么东西都有，但人不常在。

#### 人物小传

小英是这个小团体里最安静的人。

她不太说话，别人吵吵闹闹的时候她就在旁边安安静静地待着。不是不合群，是性格就这样——她更喜欢观察而不是参与。她会在大家玩捉迷藏的时候蹲在角落看一只蚂蚁搬东西，会在别人放鞭炮的时候捂着耳朵站在远处微笑，会在大家疯跑的时候一个人摘野花。

她有一种特别的能力：小动物不害怕她。别人靠近村里的流浪猫，猫跑了；她走过去，猫就让她摸。她经常在口袋里装着米粒喂麻雀，在路边捡到受伤的虫子会小心翼翼放在手心里带回家。外婆说她"前世肯定是只小鸟"。

但小英的安静不全是天性。她的家人对她期望很高——在她身上寄托了"走出这个村子"的希望。她很聪明，学习好，是那种老师会夸、邻居会羡慕的孩子。家里人觉得她将来要去县城读书、要出人头地。她没有被逼着学，但那种"你要争气"的氛围从小就在。所以她安静，一部分是因为乖巧，一部分是因为没有多余的心思去闹。和小朋友们疯玩对她来说是一件"奢侈"的事——家里人不会明说，但她总觉得玩太久不好。

和小朋友们在一起的时候，她反而觉得轻松。不用做那个"最优秀的孩子"，不用被期望，只是安安静静地待着就好。小动物不评判她、不要求她，所以她喜欢和小动物待在一起。

她也帮家里干活。不是那种大声喊出来的"我干活了"，而是默默的——择菜、扫地、洗衣服、喂鸡。她不觉得苦，因为村里所有孩子都干。但主角某次无意中握住她的手，发现那双看着干干净净的小手，掌心也有薄薄的茧。那一刻主角才意识到，小英不只是那个安安静静摘野花的女孩，她和所有人一样，在这个村子里过着一样的日子。

小凤不是她的亲姐姐，但小凤非要认她当妹妹。两人性格天差地别——小凤风风火火，她安安静静；小凤嗓门大，她说话轻声细语。但小凤最护的人就是她——谁敢欺负小英，小凤能跟人拼命。小英也知道小凤护着她，她不说，但心里很依赖这个"姐姐"。

在主角眼里，小英是"懂事"的参照。主角刚到乡下的时候很多事不懂、很多规矩不知道，小英不会说"你应该怎样"，但她的行为本身就是一种示范——帮外婆择菜、安安静静吃饭、不乱跑让大人操心。主角看到她，会不自觉地想"我也应该这样"。

#### 性格特点

- **安静内敛**：不爱说话，喜欢一个人待着。一部分是天性，一部分是压力下的退缩

- 说话慢悠悠的，不像其他孩子那么急
- 经常用"嗯"点头代替说话
- 偶尔冒出一句让人惊艳的话，因为她观察到了别人没注意到的细节

#### 与家人的关系

小英的家是一个完整的、正常的农村家庭。爸爸种地，妈妈持家，对独生女儿寄予厚望。

她喜欢和小动物待在一起，是因为小动物不评判她。蚂蚁不会问她考了多少分，小鸟不会说"你要争气"。和小朋友们在一起的时候，她也觉得轻松——阿宝出丑、小明吹牛、小凤骂人，没有人注意她，她可以安安静静地做一个不用表现的人。

#### 人物小传

小凤是这个村子里同龄人中最不好惹的小孩，也是最让人服气的小孩。

她比大多数玩伴都大一岁，这个"大一岁"在小孩子的世界里意味着天然的权威。但让她真正成为"大姐大"的不是年龄，是性格——她敢说、敢做、敢担当。别人不敢翻的墙她翻，别人不敢走的夜路她走，别人不敢顶的大人她顶。

她脾气不好，嗓门大，说话直接。阿宝说了蠢话她直接吼，小明做了决定她觉得不对就当面反驳，连二表哥她都敢呛。但她不是那种欺负人的霸道——她霸道的方向是"保护人"。谁被外村的孩子欺负了，她第一个冲上去；谁被大人冤枉了，她站出来说"不是他干的"；谁哭了她虽然嘴上说"有什么好哭的"但还是蹲下来看怎么回事。

她对小英的保护到了偏执的程度——虽然她们不是亲姐妹，但小凤单方面把小英当成了自己的妹妹。她觉得全世界都可能欺负小英，所以她必须挡在前面。小英被虫子吓到了她去把虫子踩死，小英走慢了她停下来等，小英被别人说"不爱说话""不合群"她直接怼回去"她不想跟你说话怎么了"。

这种偏执有更深的原因。小凤的爸爸常年在外地打工，一年回来一两次。妈妈一个人在家干农活带她，累得没有笑容，脾气也不好。家里不吵架——比吵架更难受的是冷。妈妈不是不爱她，只是太累了顾不上。小凤从小就知道，哭没有用，撒娇没有人接，想要什么得自己去争。所以她比别的孩子都硬气。但也正因为自己家里没有那种暖，她才会在外面拼命对别人好。她把小英当妹妹，把朋友们当家人，在外面"组建"了一个自己想要的家。

在主角眼里，小凤是"怎么在这个群体里生存"的模板——不是讨好所有人，而是有自己的立场，保护自己在意的人。但主角也会慢慢发现，小凤的"强大"背后，是一个不太幸福的家。

- **有主见**：做什么事都有自己的想法，不随大流
- **胆大**：什么都敢试，什么人都敢顶，天不怕地不怕
- **心软**：嘴上凶，但看到别人真的难过会手足无措

#### 说话方式

- 声音大，语气冲，像个小炮仗
- 喜欢用反问句："你是不是傻""这还用问"
- 命令式："别哭了""过来""你给我站住"
- 骂人的时候很凶，但骂完会偷偷关心："你没事吧……我不是说你不能那样做吗！"
- 偶尔温柔的时候反而让人不习惯

#### 与主角的关系

小凤是主角的青梅竹马之一。好感度系统会追踪主角与她的互动，不同好感度解锁不同的对话和事件，影响故事走向。

| 阶段 | 关系状态 |
|------|---------|
| **起点** | 对主角不屑一顾——"城里来的小孩能干什么" |
| **发展** | 主角做了一件让她刮目相看的事（不一定是大事，可能只是没哭），她开始认可主角 |
| **关键转折** | 某次外人欺负主角，小凤第一个冲上去，之后主角在她心里正式成了"自己人" |
| **少年阶段** | 小凤留在乡下（或也去了县城）。她和主角之间因好感度不同，重逢时的态度会有很大差异 |
| **青年阶段** | 长大后的关系走向，取决于童年阶段的好感度和关键选择 |

> 小凤是主角"被接纳"的终极标志。她认可你，整个小团体就认可你。她也是那个让主角明白"喜欢一个人，不是因为她完美，而是因为在她面前你可以不完美"的人。

#### 与小英的关系

小凤和小英不是亲姐妹，但小凤单方面把小英当成了自己的妹妹。

她觉得小英太安静了、太善良了、太容易被人欺负了。她不理解小英为什么能安安静静看一只蚂蚁看半天，但她会站在旁边等她看完。她嘴上说"你能不能走快点"，脚步却放慢了。

小凤最大的软肋就是小英。谁要是真的把小英弄哭了，小凤会发很大的火——比自己被欺负了还生气。但反过来，小英也是唯一一个能让小凤安静下来的人。小凤发脾气的时候，小英走过去拉拉她的衣角，小凤就泄气了。

> 小凤的护短不是血缘本能，是她自己选择的。这种选择比血缘更让人心疼。

#### 与阿宝的关系

小凤对阿宝是"又嫌弃又护着"。她嫌他话多、嫌他笨、嫌他总闯祸。但她也最常帮阿宝——他摔了她第一个去拉，他被外人欺负了她第一个出头。阿宝是唯一一个被小凤骂了还笑嘻嘻凑上来的人，小凤拿他没办法。

#### 与母亲的关系

小凤的妈妈不是坏人，只是一个被生活磨累了的女人。

丈夫常年在外打工，她一个人操持家里的地、喂鸡、种菜，还要带孩子。她没有舅妈那种"乐在其中"的劲头——舅妈再累也会给孩子们多炒两个菜，小凤的妈妈累到极点的时候只会说"自己去热饭"。

更深的委屈是：她生了小凤之后，婆家的脸色就变了。公公婆婆想要孙子，丈夫虽然嘴上不说，但出去打工后一年回来一两次，也像是在逃避什么。妈妈把这份怨气不自觉地转到了小凤身上——不是恨女儿，是看到女儿就想起自己的处境。

她对小凤不是不爱，是顾不上，也是不知道怎么爱。小凤摔了跤她看了一眼说"没事吧"就继续干活了；小凤考了试回来她问一句"多少分"然后就没了下文。她不打不骂，但也不亲不暖。两个人住在一个屋檐下，像合租的室友。

小凤对妈妈的感情很复杂。她隐约知道自己是"不被期待"来到这个世界上的。她不恨妈妈，知道妈妈辛苦。但她也不亲近妈妈——她不会跟妈妈说学校的事、朋友的事、心里的事。有什么话她宁愿跟小英说、跟朋友们说，也不跟妈妈说。

> 舅舅家的斗嘴是"吵完了还是家人"，小凤家的沉默是"不吵也不是家人"。

#### 与父亲的关系

小凤对爸爸的印象很模糊。他一年回来一两次，每次带回来一些零食和新衣服，待几天又走了。

爸爸回来的时候，小凤会高兴一下——但高兴不起来。她和爸爸之间太陌生了，陌生到不知道说什么。爸爸问"学习怎么样"，她说"还行"，然后就没有然后了。妈妈在旁边冷着脸，爸爸赔着笑，小凤觉得还不如爸爸不回来。

小凤隐约知道，爸爸出去打工不只是为了挣钱，也是因为不想待在这个家里。她不知道的是——当年妈妈生下她的时候，爷爷奶奶叹了一口气，爸爸沉默了很久。这些大人的事没人跟她说，但她能感觉到。她从小就知道自己是"丫头"，不是"小子"，虽然没有人明说，但那种区别对待她从小就能感觉到。

这也造就了她骨子里的要强。她不是天性要强，是被逼出来的——她要用行动证明，女儿不比儿子差。

#### 参与事件

- 带领大家决定今天玩什么的日常
- 小凤和外村孩子对峙的事件
- 保护小英不被欺负的事件
- 小凤难得温柔的一面事件（比如偷偷给小英留糖）
- 过年时小凤带头放鞭炮、比谁胆子大的事件
- 小凤和二表哥较劲的搞笑事件
- 小凤不想回家的事件（在外面玩到很晚，别人催她回家她装作没听见）
- 爸爸回来的那几天，小凤反常地安静下来的事件
- 主角第一次去小凤家、发现她家和外婆家完全不一样的事件

#### 视觉设计参考

- 穿着：袖子卷到手肘，裤子扎进袜子里，一副随时准备干架的样子
- 体态：比同龄女孩结实，跑得快，打架也不输男孩
- 表情：大部分时候皱着眉头，偶尔笑起来会露出一颗虎牙
- 动作习惯：双手叉腰说话，走路大步流星，不耐烦地甩手
- 身边常跟：小英（她走哪小英跟哪）

#### 阿婆

#### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 阿婆 |
| **年龄** | 70+ |
| **身份** | 大屋附近的邻居老人 |
| **特点** | 爱八卦、热心、身体不太好 |

#### 人物小传

阿婆是这个片区最闲的人，也是最忙的人——闲的是不用干活了，忙的是她一张嘴管着半个村的事。

她住在离外婆家不远的一间老屋里，儿女都在外面，老伴走了好几年了。她一个人住，身体不太好——腿脚不利索，走不远，天阴了膝盖就疼。但她不愿意去城里跟儿女住，说"去了就是等死，在这里好歹有人说话"。

她的日常是这样的：早上起来在门口坐着，看到路过的人就打招呼。谁家吵架了她知道，谁家来客人了她知道，谁家的鸡丢了她也知道。她不是故意打探——整个村就这么大，坐在门口什么都能听到。

她嘴碎，但不是那种讨人厌的碎。她说谁家的事不是为了嚼舌根，是因为她真的关心。她说"你舅舅又跟你舅妈吵了吧"不是看热闹，是想确认"没事吧"。她说"那个谁家的孩子又闯祸了"不是批评，是心疼。

她对外婆特别好。两个人是几十年的老邻居了，有什么好吃的都会互相送一碗。外婆忙的时候阿婆帮忙看店，阿婆生病了外婆过去照顾。她们之间不需要客套，端着碗就过去了，连门都不用敲。

对主角，阿婆是一个"永远在家"的存在。主角在村里跑来跑去，经过阿婆家门口，她永远在那儿坐着，说一句"慢点跑""饿了吧""进来喝口水"。她不参与主角的冒险，但她是主角在村里的一个锚点——不管玩到多晚、跑到多远，经过阿婆家门口就觉得快到家了。

#### 性格特点

- **嘴碎**：什么都爱说两句，但没有恶意，是真的关心
- **热心**：看到谁有需要都会帮忙，虽然自己身体也不好
- **孤独**：嘴上说不孤独，但一个人坐在门口的时候，眼神会飘远
- **要强**：不愿意给儿女添麻烦，不愿意被人当"废人"看待

#### 说话方式

- 语速慢，絮絮叨叨的，一件事能翻来覆去说三遍
- 喜欢拉住人说话："哎你等等""坐下坐下""我跟你说"
- 关心人的方式是唠叨："多穿点""别跑那么快""吃了没"
- 偶尔冒出一句很有智慧的话，因为她活了七十多年什么都见过

#### 与外婆的关系

两个人是这个村子里最亲近的老姐妹。互相送吃的、互相照应、互相吐槽儿女。她们在一起的时候像两个小女孩——虽然都七十多了，但笑起来跟年轻时候一样。

阿婆羡慕外婆有外孙来陪。她自己的孙辈都在城里，一年见不到几次。她对主角好，有一部分是因为"看到别人家的孙子，就想起自己的"。

#### 参与事件

- 日常打招呼、唠叨事件（反复出现的温馨日常）
- 主角去阿婆家喝水、吃东西的事件
- 阿婆生病了、大家担心的事件
- 阿婆讲过去的故事的事件（可能涉及外公年轻时的事）
- 阿婆的儿女打电话回来、她嘴上说"不用管我"挂了电话却坐了很久的事件

#### 视觉设计参考

- 穿着：深色旧衣服，围裙，头上包着毛巾
- 体态：瘦小、驼背、走路拄着拐杖
- 表情：笑眯眯的，说话时眼睛眯成一条缝
- 动作习惯：坐在门口的竹椅上，手里可能在择菜或者摇蒲扇
- 场景：永远在门口，像一个固定的风景

#### 村长

#### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 村长 |
| **年龄** | 50多 |
| **身份** | 村干部，阿旺阿宝的爷爷 |
| **特点** | 和蔼公正、忙碌、外公的老战友 |

#### 人物小传

村长是这个村子里说话最有分量的人。

不是因为他有多大权力——一个村长能管的事有限——是因为他做事公正，大家信他。谁家的地界有纠纷找他，谁家的年例安排找他，上面有什么政策下来也是他挨家挨户通知。他忙，但不烦，找他的人都能得到一个说法。

他年轻时和外公一起经历过"南边的仗"。两个人都不是正式军人——那个年代，有些普通人被卷进了不该他们去的事。具体发生了什么没人知道，回来之后外公变了一个人，村长倒是没怎么变——该笑笑该干活干活。但两个人之间的关系变了。以前是兄弟，现在更像是……彼此都看过对方最脆弱的样子，所以反而不太说话了。

他们见面时的模式是：递根烟，点上，坐一会儿。不聊过去，聊鱼塘、聊年例、聊天气。那种沉默不是尴尬，是不需要说。

村长对阿旺阿宝很疼爱。他的大儿子（阿旺阿宝的爸爸）在外面搞工程，常年不回来，赚了钱但顾不上家。村长管不了儿子——就像外公管不了舅舅一样。上一代的人面对下一代，都有这种无力感。但他对两个孙子是真的好，过年的时候给的压岁钱最多，阿宝闯了祸他也不会真骂。

在主角眼里，村长是一个"大人的世界"的窗口。他和外公的关系、他对大儿子的态度，都是主角长大后才会慢慢理解的事。

#### 性格特点

- **公正**：处理事情不偏不倚，大家都信服
- **和蔼**：对孩子们很慈祥，说话笑眯眯的
- **有城府**：不是坏的那种城府，是见过世面的人才有的沉稳
- **不提过去**：关于那场仗，跟外公一样，一个字不提

#### 说话方式

- 语气平稳，像在说家常
- 对孩子们说话很温柔："慢慢来""不着急"
- 处理村务时干脆利落："行""可以""这事我来"
- 提到外公时话会少一些，但眼神里有东西

#### 与外公的关系

两个人是过命的交情，但回来之后各走各的路。外公选择沉默地守鱼塘，村长选择融入村子当了干部。没有谁对谁错，只是面对同一件事的不同方式。

他们很少串门，但遇到大事——台风来了、鱼塘出事——村长会出现在鱼塘边的小瓦房里。外公也不意外，就像知道他会来一样。

> 外公和村长的关系，是这个游戏里"大人之间不说出口的友情"的代表。

#### 与阿旺阿宝的关系

村长对两个孙子的疼爱是发自内心的。阿旺阿宝经常去爷爷家蹭饭——因为自己家太冷清了。爷爷家虽然也不大，但有热饭、有笑声、有人问"今天在学校怎么样"。

村长心疼大儿子顾不上家，但说不出口。他能做的就是对两个孙子好一点，再好一点。

#### 与那场仗的关系

游戏里不会明说那是什么仗、在哪里、发生了什么。只通过碎片让玩家拼凑：
- 外公和村长是"一起回来的"
- 回来后外公变了，村长没怎么变
- 两人之间有一种不用说的默契
- 偶尔有老人聊天提到"他们那一批人"，但很快岔开话题

> 这是游戏内容模糊化规则的体现：不直接使用真实事件名称，保持真实感底色但不被具体历史绑定。

#### 参与事件

- 年例组织事件（村长是主要组织者）
- 处理村务的日常事件
- 主角无意中听到村长和外公对话的事件
- 台风天村长出现在鱼塘边的事件
- 过年给压岁钱的事件
- 阿旺阿宝去爷爷家蹭饭的温馨事件

#### 视觉设计参考

- 穿着：比普通村民整洁一点，白衬衫+西裤（开会时穿），平时也是旧衣服
- 体态：中等身材，微微发福，但精神很好
- 表情：大部分时候是温和的微笑，处理事情时严肃
- 动作习惯：抽烟，说话时爱用手比划
- 场景：经常出现在村委会、村道上、或者挨家挨户串门

#### 小明妈

#### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 小明妈、亚明他妈 |
| **年龄** | 30多 |
| **身份** | 小明的母亲，独自带大小明 |
| **特点** | 沉默、坚韧、不抱怨 |

#### 人物小传

小明妈是这个村子里最沉默的女人之一。

丈夫走后，她一个人扛起了整个家。种地、干家务、养孩子，日复一日。她不跟人诉苦，不跟人哭，别人提起她丈夫的事她就岔开话题。不是因为还痛——是痛过了，已经不说了。

她对小明的方式很特别：不溺爱，不打骂，但也不温柔。她让小明自己学着干活——劈柴、挑水、修东西。不是因为她狠心，是因为她知道"我不能陪他一辈子，他得学会自己照顾自己"。

她和舅妈是两种完全不同的母亲。舅妈是温暖的、话多的、把爱挂在脸上的；小明妈是冷的、话少的、把爱藏在行动里的。但两个人有一个共同点：她们都在用自己的方式撑着这个家。

村里人都敬重她。不是因为她做了什么了不起的事，是因为她什么都做了、什么都扛了、一个字都不说。

> 小明的坚强来自她。她没有给小明一个完整的家，但给了他一副硬骨头。

#### 性格特点

- **沉默**：不爱说话，不是不想说，是说了也没用
- **坚韧**：什么都自己扛，不求人
- **务实**：不浪漫、不幻想，只想过好今天的日子
- **有尊严**：穷但不卑微，苦但不卖惨

#### 说话方式

- 话很少，能用一个字说完的不用两个字
- 对小明说的最多的是："回来吃饭""早点睡""别给人添麻烦"
- 偶尔被小明惹生气了也只是说一句："你能不能让人省点心"
- 她不说"我爱你"，但每天的饭菜都是热的

#### 与小明的关系

母子之间的爱不用语言表达。小明帮妈妈干活不说"我来帮你"，直接上手；妈妈给小明做饭不说"多吃点"，把菜往他碗里拨。

两个人之间最安静也最默契。他们不需要说的话，别人家说了一箩筐也说不完。

> 主角看到小明和妈妈的相处方式，会想起自己的爸妈——也许开始理解"不在身边不代表不爱"。

#### 参与事件

- 主角去小明家、看到阿姨沉默地做饭的日常事件
- 小明帮妈妈干农活、主角默默加入的事件
- 过年时小明家安安静静、妈妈给小明做了一桌好菜的事件
- 小明妈和舅妈偶尔聊天的事件（两个女人的互相理解）

#### 视觉设计参考

- 穿着：朴素的深色衣服，袖子卷着，围裙上沾着灰
- 体态：不胖不瘦，但看起来比实际年龄老一些
- 表情：大部分时候面无表情，偶尔看到小明会有一闪而过的温柔
- 动作习惯：干活很利索，走路很快，不浪费时间
- 场景：永远在干活——在地里、在厨房、在院子里

#### 小凤妈

#### 基本信息

| 项目 | 内容 |
|------|------|
| **称呼** | 小凤妈、彩凤她妈 |
| **年龄** | 30多 |
| **身份** | 小凤的母亲，丈夫常年在外打工 |
| **特点** | 疲惫、烦躁、不是不爱是顾不上 |

#### 人物小传

小凤妈不是坏人，只是一个被生活磨累了的女人。

丈夫常年在外打工，一年回来一两次。她一个人操持家里的地、喂鸡、种菜，还要带孩子。她没有舅妈那种"乐在其中"的劲头——舅妈再累也会给孩子们多炒两个菜，小凤妈累到极点的时候只会说"自己去热饭"。

更深的委屈是：生了小凤之后，婆家的脸色就变了。公公婆婆想要孙子，丈夫虽然嘴上不说，但出去打工之后回来得越来越少。她把这份怨气不自觉地转到了小凤身上——不是恨女儿，是看到女儿就想起自己的处境。

她和小明妈都是独自带孩子的女人，但两个人的反应完全不同。小明妈把苦咽下去了，变成了一种沉默的力量；小凤妈的苦咽不下去，变成了一种说不清的烦躁。她不打小凤，不骂小凤，但也不亲近小凤。两个人住在同一个屋檐下，像两个合租的人。

她不是不爱小凤。小凤生病了她也着急，小凤摔了跤她也心疼。但她不会表达——她的温柔被生活磨掉了，只剩下一种本能的、沉默的、不被察觉的关心。

> 舅舅家的斗嘴是"吵完了还是家人"，小凤家的沉默是"不吵也不是家人"。小凤妈不是不想吵，是连吵架的力气都没有了。

#### 性格特点

- **疲惫**：不是身体累，是心累。生活没有给她喘息的机会
- **烦躁**：容易不耐烦，对小凤的回应经常是"别烦我"
- **沉默**：不是不想说，是说了也没人听。丈夫不在，婆婆不亲
- **有爱但表达不出**：她对小凤的关心藏在行动里，但从不说出口

#### 说话方式

- 语气经常是不耐烦的："自己去""你自己不会弄吗""别来烦我"
- 但偶尔也会有一句温柔的："多穿点""别太晚回来"
- 这种温柔太少太少，所以小凤每次听到都会愣一下

#### 与小凤的关系

两个人之间最大的问题不是恨，是距离。

小凤不恨妈妈，知道妈妈辛苦。但她也不亲近妈妈——不会跟妈妈说学校的事、朋友的事、心里的事。妈妈不问，她就不说。妈妈偶尔问一句，她也觉得不是真心想听，只是随口一问。

小凤在外面组建了自己"家"——把小英当妹妹、把朋友们当家人。她不是故意要离开妈妈的世界，是妈妈的世界里没有她的位置。

#### 与小明妈的关系

两个女人都是独自带孩子，但活法完全不同。

小明妈沉默而有力，小凤妈沉默而无力。她们偶尔在村道上遇到，会点点头、说两句闲话，但从不深聊。不是不想聊，是聊了又怎样——各自的苦不是说出来就能解决的。

> 如果说小明妈是"把苦变成了力量"，小凤妈就是"被苦压着没有翻身"。没有对错，只是不同的人面对同样的处境，有不同的承受方式。

#### 参与事件

- 主角去小凤家、发现她妈妈冷冰冰的事件
- 小凤和妈妈之间冷淡对话的日常事件
- 小凤妈难得温柔一面的事件（比如小凤生病了她守了一夜）
- 过年丈夫回来那几天、家里气氛微妙变化的事件

#### 视觉设计参考

- 穿着：朴素的旧衣服，不太讲究
- 体态：比同龄人显老，脸上有操劳的痕迹
- 表情：大部分时候是沉着脸的，偶尔叹一口气
- 动作习惯：干活很快但很沉默，不抬头、不闲聊
- 场景：在地里干活、或者在厨房里一个人做饭

#### 村民群像

这些角色不需要详细档案，他们是村庄氛围的一部分。玩家在探索时会遇到他们，发生简短的互动。

---

#### 花婶

- **身份**：村口小卖部老板娘
- **年龄**：40多
- **特点**：嗓门大、热情、消息灵通
- **场景**：守着村口一间铁皮棚小卖部，卖烟酒零食日用品
- **作用**：村里的信息集散地。谁家来客人了、谁跟谁吵架了、镇上有什么新鲜事，她都知道
- **说话方式**："来坐啊""吃点什么""我跟你说那个谁家……"
- **事件**：主角去买东西、听到花婶跟别人聊八卦的日常事件

---

#### 医生伯伯

- **身份**：村卫生站医生
- **年龄**：50多
- **特点**：话不多、医术一般但态度好
- **场景**：卫生站在村子中间，一间老房子，药柜上摆着各种药
- **作用**：日常小病事件。村里人头疼脑热都来找他，严重的才去镇上
- **说话方式**："张嘴看看""没事，吃点药就好了""这个要去镇上看"
- **事件**：主角感冒了去看病的事件；阿宝摔跤了被抬过去的事件

---

#### 摩托车佬（老陈）

- **身份**：跑三轮摩托车载客的中年人
- **年龄**：40多
- **特点**：话多、爱吹牛、但人不坏
- **场景**：停在村口等客，或者在路上跑
- **作用**：赶集场景。村里人去镇上赶集经常坐他的车
- **说话方式**："去哪里？""上来上来""今天镇上热闹得很"
- **事件**：一起坐摩托车去赶集的事件

---

#### 阿德叔

- **身份**：养鱼的农民，帮外公干活
- **年龄**：40多
- **特点**：老实、勤快、不太说话
- **场景**：经常在鱼塘边出现，帮外公喂鱼、清理鱼塘
- **作用**：日常在鱼塘边能遇到的人，跟主角说两句闲话
- **说话方式**："你外公在那边""小心别滑下去""这条鱼不错"
- **事件**：帮阿德叔喂鱼的日常事件

---

#### 三婆

- **身份**：村里的老人，和阿婆关系好
- **年龄**：70+
- **特点**：耳朵不太好使，但身体比阿婆好
- **场景**：经常和阿婆一起坐在门口聊天
- **作用**：两个老人聊天是村里一道风景——一个大声说一个大声听
- **说话方式**："啊？你说什么？""哦哦哦我知道了"
- **事件**：路过时听到两个阿婆聊天的搞笑事件

---

#### 阿辉

- **身份**：邻家小哥，在茂名市区读高中
- **年龄**：17
- **特点**：假期才回乡下，带回县城的新鲜事
- **场景**：假期在村里晃，平时不在
- **作用**：带来外界信息。他知道县城有什么新鲜游戏、新鲜歌、新鲜事
- **说话方式**："你们还在玩这个？""县城那边都玩那个了"
- **事件**：假期回来给村里孩子讲县城新鲜事的事件

---

#### 小英爸

- **身份**：小英的父亲，普通农民
- **年龄**：35左右
- **特点**：老实、话少、对女儿期望高
- **场景**：在地里干活，或者在家里
- **作用**：小英家庭氛围的具象化。他不说"你要争气"，但看向小英的眼神里全是期望
- **说话方式**："好好学""考了多少分""要争气"
- **事件**：偶尔在小英家遇到她爸爸的事件

---

#### 设计原则

这些村民群像的共同特点：
- **不抢戏**：他们是背景，不是主线
- **有温度**：每个人都代表乡村生活的一个切面
- **可互动**：主角经过时能触发简短对话，增加沉浸感
- **有记忆点**：每个角色有一两个鲜明特征（嗓门大、耳朵背、爱吹牛……）

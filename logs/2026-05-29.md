### 2. CourtyardStageBuilder 生成器

新增 `scripts/scenes/courtyard_stage_builder.gd`，院落专用视觉生成器，和 InteriorStageBuilder 同级。支持五种元素类型：

| 类型 | 用途 |
|------|------|
| `rect` | 矩形色块（地面、墙面） |
| `line` | 线条（透视线、电线） |
| `ellipse` | 椭圆（井口、石桌、接触阴影） |
| `poly` | 多边形（透视地面梯形） |
| `sprite` | PNG 素材（主屋、老屋） |

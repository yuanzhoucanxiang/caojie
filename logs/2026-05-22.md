## 犯的错误

### Shader 初始就显示遮罩（3次修复）
- 原因：`step(eff, radius)` 中 eff 可以为负值，step(负值, 0) = 1
- 修复：改用 `coverage = (0.5 - edge_dist) + n * 0.15`，保证 coverage >= 0
- 避免：shader 中 step(a,b) 注意 a<=b 返回 1

### 水彩过渡不明显/闪烁
- 原因：参数太小、过渡太快、easing 曲线突变
- 修复：增大参数、加长过渡时间、用 TRANS_SINE + EASE_IN_OUT
- 避免：特效参数先做大再缩小

### 褪色遮罩用了纯白色
- 原因：Color(1, 1, 1, 0) 违反美术规则
- 修复：改用场景主色调 transition_color
- 避免：美术规则写入 CLAUDE.md，每写新代码前检查

### Tabby PATH 未生效
- 原因：修改 PATH 后当前终端窗口读不到
- 解决：新开终端窗口（或重启 Tabby）
